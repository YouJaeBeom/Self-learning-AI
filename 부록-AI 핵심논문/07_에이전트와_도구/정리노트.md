# T07 정리노트 — 계보와 요약

## 계보: 루프에서 측정까지

ReAct가 생각→행동→관찰 루프로 에이전트의 골격을 세우자, 그 루프 안의 행동(도구)을 어떻게 잘하느냐가 과제가 되었습니다. Toolformer가 도구 호출을 자기지도로 학습해 도구 사용을 모델에 내재화했고, Reflexion이 실패를 말로 복기해 가중치 없이 개선하는 길을 더했습니다. 그리고 SWE-bench가 "이 에이전트들이 정말 일을 해내는가"를 실제 GitHub 이슈로 측정해, 분야 전체의 방향과 속도를 만들었습니다.

| 세대 | 논문 | 한 일 | 남긴 과제 |
|---|---|---|---|
| 바이블 | ReAct (2022/23) | 생각 + 행동 루프 | 오류 누적·맥락 손실 |
| 바이블 | Toolformer (2023) | 도구 호출 자기학습 | 호출 판단의 불완전 |
| 최신 | Reflexion (2023) | 언어 자기반성으로 개선 | 반성 자체의 오류 |
| 최신 | SWE-bench (2023) | 현실적 능력 측정 | 오염·굿하트 |

## 한 줄 요약 (논문별)

| 논문 | 분류 | 한 줄 핵심 |
|---|---|---|
| ReAct (Yao et al., 2022/23, ICLR 2023, arXiv:2210.03629) | 바이블 | 생각→행동→관찰 — 에이전트의 기본 루프 |
| Toolformer (Schick et al., 2023, NeurIPS, arXiv:2302.04761) | 바이블 | 유용했던 호출만 추려 도구 사용을 자기지도 학습 |
| Reflexion (Shinn et al., 2023, NeurIPS, arXiv:2303.11366) | 최신 | 실패를 말로 복기해 가중치 없이 개선 |
| SWE-bench (Jimenez et al., 2023, ICLR 2024, arXiv:2310.06770) | 최신 | 실제 GitHub 이슈로 에이전트를 측정 |

서지정보는 검증된 논문 마스터에서 그대로 옮겼습니다.

## 에이전트를 이루는 네 조각

| 조각 | 담당 논문 | 역할 |
|---|---|---|
| 루프 | ReAct | 추론과 행동의 순환 |
| 도구 | Toolformer | 외부 능력 빌리기 |
| 기억·개선 | Reflexion | 경험에서 배우기 |
| 평가 | SWE-bench | 실제 능력 측정 |

## 연결

- 직관은 기초편 6단계(에이전트), 구조는 심화편 S9(에이전트와 시스템)에서.
- 자기반성(Reflexion)은 T06 Self-RAG와, 측정(SWE-bench)은 T10 평가와 통합니다.
