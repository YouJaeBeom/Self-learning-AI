# T06 · 검색·지식·환각 — 모델 바깥에서 답을 찾다

LLM은 모르는 것도 그럴듯하게 지어냅니다(환각). 게다가 학습 시점 이후의 지식은 알지 못합니다. 이 두 한계의 뿌리는 같습니다. 지식이 가중치 속에 고정되어 있다는 것입니다. 이 테마는 그 한계를 모델 바깥의 검색으로 푸는 계보를 따라갑니다. 검색과 생성을 한 틀로 묶은 RAG에서 출발해, 검색을 정밀하게 만들고, 검색을 똑똑하게 쓰고, 마침내 지식을 구조로 정리해 전역 질문까지 답하는 데로 나아갑니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | RAG (Lewis et al., 2020) | 검색 + 생성을 한 틀로 묶어 'RAG'라는 패러다임을 만듦 |
| 바이블 | DPR (Karpukhin et al., 2020) | 학습된 임베딩(dense)으로 키워드 검색(BM25)을 넘어섬 |
| 최신 | ColBERT (Khattab & Zaharia, 2020) | 토큰 단위 늦은 상호작용으로 정밀도와 효율을 동시에 |
| 최신 | Self-RAG (Asai et al., 2023) | 모델이 언제 검색할지·근거가 맞는지를 스스로 비평 |
| 최신 | GraphRAG (Edge et al., 2024) | 지식 그래프로 문서 전체를 종합하는 질문에 답 |

## 이 테마를 관통하는 한 문장

RAG가 검색과 생성을 잇는 틀을 열자 검색을 잘하는 법이 과제가 되었고(DPR·ColBERT), 그다음은 검색을 똑똑하게 쓰는 법(Self-RAG)과 지식을 구조로 정리하는 법(GraphRAG)으로 초점이 옮겨 갑니다. 환각을 줄이는 길이 "더 큰 모델"이 아니라 "더 나은 검색"에 있다는 관점을 이 계보가 보여 줍니다.
