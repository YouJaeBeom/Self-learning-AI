# T06 정리노트 — 계보와 요약

## 계보: 검색을 잘하고, 똑똑하게 쓰기

RAG가 "검색 + 생성"이라는 틀을 열자, 자연히 검색을 잘하는 법이 과제가 되었습니다. DPR이 단어가 아닌 의미로 찾는 dense 검색의 토대를 놓았고, ColBERT가 토큰 단위 늦은 상호작용으로 정밀함과 속도를 절충했습니다. 이후 초점은 검색을 더 똑똑하게 쓰는 법으로 옮겨, Self-RAG가 "언제 검색하고 근거가 맞는지"를 모델이 스스로 비평하게 했고, GraphRAG가 조각 검색을 넘어 지식 그래프로 전역 종합까지 나아갔습니다.

| 세대 | 논문 | 한 일 | 남긴 과제 |
|---|---|---|---|
| 바이블 | RAG (2020) | 검색 + 생성 틀 | 검색 품질이 전부 |
| 바이블 | DPR (2020) | 의미(dense) 검색 | 단일 벡터의 거침 |
| 최신 | ColBERT (2020) | 토큰 단위 정밀 검색 | 색인 비용 |
| 최신 | Self-RAG (2023) | 적응형·자기비평 검색 | 자기평가 오류 |
| 최신 | GraphRAG (2024) | 그래프로 전역 종합 | 구축 비용 |

## 한 줄 요약 (논문별)

| 논문 | 분류 | 한 줄 핵심 |
|---|---|---|
| RAG (Lewis et al., 2020, NeurIPS, arXiv:2005.11401) | 바이블 | 가중치 기억 + 외부 검색 = 오픈북 패러다임 |
| DPR (Karpukhin et al., 2020, EMNLP, arXiv:2004.04906) | 바이블 | 듀얼 인코더로 의미가 통하면 찾는 dense 검색 |
| ColBERT (Khattab & Zaharia, 2020, SIGIR, arXiv:2004.12832) | 최신 | 토큰별 벡터 + MaxSim 늦은 상호작용 |
| Self-RAG (Asai et al., 2023, ICLR 2024 Oral, arXiv:2310.11511) | 최신 | 반성 토큰으로 검색·근거를 스스로 비평 |
| GraphRAG (Edge et al., 2024, arXiv:2404.16130) | 최신 | 지식 그래프 + 커뮤니티 요약으로 전역 질문 |

서지정보는 검증된 논문 마스터에서 그대로 옮겼습니다.

## 검색 방식 비교

| 방식 | 표현 | 강점 | 약점 |
|---|---|---|---|
| BM25 | 단어 빈도 | 정확 매칭, 빠름 | 어휘 불일치 |
| DPR | 단일 의미 벡터 | 의미 검색 | 세부 손실 |
| ColBERT | 토큰별 벡터 | 정밀 + 효율 | 색인 비용 |

## 연결

- 직관은 기초편 6단계(RAG), 시스템 설계는 심화편 S8(검색과 지식)에서.
- 자기비평(Self-RAG)은 다음 테마 T07 에이전트의 자기반성(Reflexion)과 사상이 통합니다.
