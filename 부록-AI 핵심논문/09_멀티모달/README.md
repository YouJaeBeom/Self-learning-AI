# T09 · 멀티모달 — 텍스트 너머, 보고 듣는 모델

언어 모델은 글만 압니다. 그런데 세상은 이미지, 음성, 영상으로 가득합니다. 텍스트와 다른 형태의 데이터를 하나의 모델이 다루게 만드는 것이 멀티모달의 과제입니다. 이 테마는 그 계보를 따라갑니다. 이미지와 텍스트를 한 공간에 정렬한 CLIP에서 출발해, 동결 LLM에 비전을 붙이고, 그 결합을 단순화하고, 마침내 모든 모달리티를 통합하거나 고해상도까지 정밀하게 보는 최신 모델로 이어집니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | CLIP (Radford et al., 2021) | 이미지·텍스트를 한 공간에 대조 정렬 |
| 바이블 | Flamingo (Alayrac et al., 2022) | 동결 LLM + 비전을 cross-attention으로 결합 |
| 최신 | LLaVA (Liu et al., 2023) | 단순 연결 + 시각 instruction tuning |
| 최신 | Chameleon (Meta FAIR, 2024) | 모든 모달을 토큰화한 early-fusion |
| 최신 | Qwen2-VL (Qwen, 2024) | 임의 해상도 + M-RoPE |

## 이 테마를 관통하는 한 문장

멀티모달의 큰 줄기는 정렬 → 결합 → 통합으로 흘러왔습니다. CLIP이 이미지·텍스트를 정렬해 공용어를 만들자, Flamingo가 동결 LLM에 비전을 결합해 보고 말하는 VLM을 열었고, LLaVA가 그 결합을 단순화해 대중화했습니다. 이후 모든 모달을 하나로 합치는 통합형(Chameleon)과 모듈형의 지각력을 끌어올리는 방향(Qwen2-VL)으로 갈라집니다.
