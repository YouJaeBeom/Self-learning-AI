# 07 연합학습 (논문편) — 정리노트

## 계보: 발상에서 현실로

2017년 FedAvg가 "데이터를 옮기지 않고 로컬 학습 + 가중평균"으로 연합학습 분야를 창시했습니다. 현실의 이질성을 만나자 2020년 FedProx가 proximal term으로 client drift와 straggler를 잡았고, 같은 해(2017) Secure Aggregation이 상쇄 마스크로 서버조차 개별 업데이트를 못 보게 했습니다. 단일 글로벌 모델의 한계가 드러나며 개인화로 나아가, Ditto(ICML 2021)가 정규화 개인화로 공정성과 견고성을, FedBN(ICLR 2021)이 BN 로컬 유지로 feature shift를 풀었습니다. 그리고 2024년 FederatedScope-LLM이 PEFT로 거대 LLM을 연합 파인튜닝하는 길을 열었습니다.

한 문장으로: **창시(FedAvg) → 이질성(FedProx)·프라이버시(Secure Agg) → 개인화(Ditto·FedBN) → 파운데이션 모델(FederatedScope-LLM).**

| 단계 | 논문 | 분류 | 계보적 역할 |
|---|---|:---:|---|
| 1 | FedAvg (2017) | 바이블 | 연합학습 창시, 가중평균 |
| 2 | FedProx (2020) | 바이블 | proximal term으로 이질성 대응 |
| 3 | Secure Aggregation (2017) | 바이블 | 개별 은닉, 합계만 노출 |
| 4 | Ditto (ICML 2021) | 최신 | 정규화 개인화 — 공정·견고 |
| 5 | FedBN (ICLR 2021) | 최신 | BN 로컬 유지 — feature shift |
| 6 | FederatedScope-LLM (2024) | 최신 | PEFT 연합 LLM 파인튜닝 패키지 |

## 한 줄 요약 (논문별)

| 논문 | 분류 | 한 줄 핵심 |
|---|:---:|---|
| FedAvg (McMahan et al., 2017) | 바이블 | 로컬 여러 스텝 + 데이터 양 가중평균 |
| FedProx (Li et al., 2020) | 바이블 | 글로벌에 묶는 proximal term으로 non-IID·straggler |
| Secure Aggregation (Bonawitz et al., 2017) | 바이블 | 상쇄 마스크로 합계만, dropout 복구 |
| Ditto (Li et al., ICML 2021) | 최신 | 글로벌에 묶인 개인화 모델 — 공정·견고 |
| FedBN (Li et al., ICLR 2021) | 최신 | BN만 로컬 유지로 feature non-IID 흡수 |
| FederatedScope-LLM (Kuang et al., 2024) | 최신 | 작은 어댑터(PEFT)만 교환하는 연합 LLM |

서지: FedAvg(AISTATS 2017, arXiv:1602.05629), FedProx(MLSys 2020, arXiv:1812.06127), Secure Aggregation(CCS 2017, DOI:10.1145/3133956.3133982), Ditto(ICML 2021, 게재처 기준 표기), FedBN(ICLR 2021, 게재처 기준 표기), FederatedScope-LLM(KDD 2024, arXiv:2309.00363). Ditto·FedBN은 게재처로만 표기하고 arXiv 식별자는 단정하지 않습니다.

## 세 가지 도전과 해법

| 도전 | 증상 | 해법 |
|---|---|---|
| 이질성 | client drift, straggler | FedProx, Ditto, FedBN |
| 프라이버시 | 업데이트 누수 | Secure Aggregation |
| 규모 | LLM 통신 폭발 | FederatedScope-LLM(PEFT) |
