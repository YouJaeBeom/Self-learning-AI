# 피팅 없는 정형 예측 — TabPFN

> **Accurate predictions on small data with a tabular foundation model (TabPFN v2)** — Hollmann et al., 2025, Nature 637 (DOI:10.1038/s41586-024-08328-6; 선행 ICLR 2023 arXiv:2207.01848)
> 원문: https://doi.org/10.1038/s41586-024-08328-6 · 분류: 최신

20년 가까이 표 데이터의 왕좌는 그래디언트 부스팅 트리였습니다. Random Forests, XGBoost, LightGBM, CatBoost가 그 계보입니다. 정형 딥러닝이 여러 번 도전했지만 일관된 우위를 보이지는 못했죠. TabPFN은 완전히 다른 길로 그 통념을 흔들었습니다. 데이터마다 모델을 새로 학습하지 않습니다. 한 번 사전학습된 트랜스포머가 데이터를 읽고 즉석에서 예측합니다. 그리고 그 결과가 Nature에 실렸습니다. "데이터마다 피팅"이라는 패러다임과 "작은 표 데이터는 부스팅"이라는 통념을 함께 흔든 사건입니다.

## 배경: 작은 데이터와 매번 새로 학습하는 비용

정형 문제에는 두 가지 현실적 제약이 있습니다. 첫째, 작은 데이터가 흔합니다. 의료, 과학 실험, 비즈니스 데이터는 수천 행에 그치는 경우가 많습니다. 딥러닝은 데이터가 적으면 제 실력을 못 내고, 트리 모델도 적절한 튜닝이 필요합니다. 둘째, 매번 새로 학습하는 비용입니다. 부스팅은 데이터셋마다 처음부터 피팅하고 튜닝해야 합니다. 언어와 이미지에는 한 번 사전학습해 여러 과제에 쓰는 파운데이션 모델이 있는데, 정형데이터에는 그런 것이 없었습니다.

> 핵심 질문: 언어·이미지처럼, 정형데이터에도 한 번 학습해 두면 새 데이터를 바로 푸는 파운데이션 모델을 만들 수 있을까? 그것도 작은 데이터에서 부스팅을 능가하면서?

## 핵심 아이디어: 맥락 내 학습

발상의 전환은 in-context learning(맥락 내 학습)입니다. 수백만 개의 합성 정형 데이터셋으로 트랜스포머를 미리 사전학습해 둡니다. 실제 문제에서는 가중치를 고치지 않고, 학습 데이터(행 전부)를 프롬프트처럼 통째로 입력해, 테스트 행의 답을 단 한 번의 순전파로 예측합니다.

모델은 "어떻게 학습하는가"를 미리 배워 둡니다. 다양한 가짜 데이터셋에서 "이런 데이터엔 이런 함수가 맞더라"는 패턴을 사전학습으로 익힙니다. 실전에서는 기울기 업데이트가 없습니다. 학습 데이터는 모델을 고치는 대신 입력의 맥락이 됩니다. LLM이 예시 몇 개만 보고 새 과제를 풀듯이 말입니다.

> 비유 — 베테랑과 신입. 부스팅은 새 직장마다 처음부터 일을 배우는 신입입니다(데이터셋마다 재학습). TabPFN은 수많은 회사를 거친 베테랑이라, 새 회사의 인수인계 자료(학습 데이터)를 한 번 쭉 읽고 바로 일을 시작합니다. 따로 연수받을(피팅할) 필요가 없습니다.

이론적 뿌리는 PFN(Prior-data Fitted Networks)입니다. "사전(prior)에서 뽑은 데이터로 학습하면, 모델이 베이즈 추론을 근사하게 된다"는 발상입니다. 그래서 이름이 Tabular PFN입니다.

## 어떻게 작동하나

```
[사전학습, 한 번만]
 수백만 개 합성 데이터셋 생성(다양한 구조)
   → 트랜스포머가 "데이터 → 예측" 매핑 자체를 학습

[실제 사용, 학습 없음]
 새 문제의 (학습 행 전부 + 테스트 행)을 한꺼번에 입력
   → 순전파 1회 → 테스트 행 예측        ← 기울기 업데이트 없음
```

학습이 곧 추론 한 번입니다. 데이터셋마다 따로 피팅하거나 튜닝하지 않아 사실상 즉시 예측합니다. v2(2025, Nature)는 더 큰 표, 더 많은 특성, 범주형과 결측을 다루도록 확장했고 회귀까지 지원합니다. 보고된 결과는 작은 규모에서 중간 규모의 정형데이터에서 잘 튜닝한 부스팅과 대등하거나 능가하며, 게다가 튜닝이 불필요해 훨씬 빠르게 결과를 냈다는 것입니다. 이것이 부스팅 우위 통념을 흔든 핵심이고, Nature 게재의 근거입니다.

| 항목 | 부스팅 | TabPFN |
|---|---|---|
| 학습 | 데이터셋마다 피팅 | 사전학습 한 번 |
| 사용 시점 | 튜닝 필요 | 순전파 1회 |
| 강점 영역 | 대용량·고도 튜닝 | 작은~중간 규모 |

## 임팩트: 무엇을 바꿨나

- **정형데이터에 파운데이션 모델 시대를 열었습니다.** "데이터마다 피팅" 패러다임에 "한 번 사전학습 후 맥락으로 즉석 예측"이라는 대안을 제시했습니다.
- **부스팅 우위 통념을 흔든 Nature 게재**입니다. 표 데이터 ML의 오랜 상식("작은 정형은 부스팅")에 정면으로 도전한 사건으로 받아들여집니다.
- **튜닝 없는 강력한 베이스라인**입니다. 작은 데이터와 빠른 프로토타이핑에서 새로운 1순위 선택지가 되었습니다.
- LLM의 in-context learning을 정형데이터로 가져온, 패러다임 교차의 상징적 사례입니다.

## 한계와 주의점

- **"부스팅을 끝냈다"가 아닙니다.** 보고된 강점은 주로 작은~중간 규모에 집중됩니다. 대용량, 초고차원, 고도로 튜닝한 환경에서는 부스팅이 여전히 강력합니다.
- **맥락 크기 제약**이 있습니다. 학습 데이터를 입력에 통째로 넣으므로, 행과 특성이 매우 많으면 메모리와 맥락 한계에 부딪힙니다(버전마다 권장 규모가 다름).
- **합성 prior 의존**입니다. 사전학습에 쓴 합성 데이터의 가정에서 멀리 벗어난 분포에서는 성능을 장담하기 어렵습니다.
- **벤치마크는 조건부**입니다. "능가했다"는 특정 규모와 프로토콜에서의 결과이므로, 인용 수치를 단정하기보다 경향으로 읽는 것이 좋습니다.

## 한 줄 요약

> TabPFN은 수많은 합성 데이터셋으로 사전학습한 트랜스포머가 가중치를 고치지 않고 학습 데이터를 맥락으로 읽어 순전파 한 번에 예측하는 정형 파운데이션 모델이다. "데이터마다 피팅"과 "작은 표는 부스팅"이라는 두 통념을 함께 흔들어 Nature에 실렸다.
