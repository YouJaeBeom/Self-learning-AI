# 02 · 군집화 — 바이블에서 최신까지

정답표 없이 데이터를 무리로 가르는 일을, 어떤 원리로 풀어 왔을까요. 이 테마는 "빽빽한 곳"(밀도)에서 "연결의 결"(스펙트럴)을 거쳐, "묶기 좋은 표현을 스스로 배우기"(딥)까지 군집화의 큰 줄기를 원전으로 잇습니다. 핵심은 "무엇을 닮음으로 볼 것인가"라는 질문이 주어진 거리에서 연결 구조로, 다시 학습된 표현으로 진화한 이야기입니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | DBSCAN (1996) | 밀도로 임의형태 군집 + 노이즈 분리, k 불필요 |
| 바이블 | Spectral Clustering (2001/02) | 라플라시안 고유벡터로 군집 + 이론적 보장 |
| 최신 | HDBSCAN (2013) | 밀도를 계층으로 + stability로 다밀도에 강건 |
| 최신 | Contrastive Clustering (2021) | 인스턴스·클러스터 이중 대조로 end-to-end 딥 군집 |
| 최신 | SCAN (2020) | 자기지도 표현 + 군집의 2단계 분리, 비지도 분류 SOTA 갱신 |

## 이 테마를 관통하는 한 문장

밀도(DBSCAN)와 연결성(스펙트럴)이라는 두 고전이 거리와 구조로 군집을 정의했고, 그 위에서 밀도를 계층화(HDBSCAN)하거나 표현 자체를 학습(Contrastive Clustering·SCAN)하는 딥 시대로 옮겨 왔습니다. "무엇을 닮음으로 볼 것인가"가 주어진 거리에서 연결 구조로, 끝내 학습된 표현으로 진화한 것입니다.
