# 08 · 컨티뉴얼 러닝 — 논문편 (바이블 → 최신)

순차적으로 배우면서 파국적 망각을 어떻게 막을까요. 신경망은 새 작업을 배우면 옛 작업을 급격히 잊는데, 사람처럼 평생 배우며 쌓아 가는 모델을 만들려면 이 망각을 막아야 합니다. 가중치로 막나, 데이터로 막나, 출력으로 막나? 그리고 사전학습 시대엔 어떻게 달라지나? 이 테마는 망각 방어의 세 갈래(정규화·재생·증류)와 사전학습이 연 새 길까지 다섯 편의 논문으로 따라갑니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | EWC (2017) | 피셔 정보로 중요 가중치를 고정(정규화) |
| 바이블 | iCaRL (2017) | exemplar+증류+NME로 클래스 증분 학습(재생) |
| 바이블 | LwF (2016/2018) | 옛 데이터 없이 증류로 망각 완화 |
| 최신 | L2P (2022) | 고정 백본 + prompt pool 선택 |
| 최신 | CLEAR / Experience Replay (2019) | replay + behavioral cloning(RL) |

보조·대안: GEM(2017)·DER++(2020)·DualPrompt(2022)는 본문에서 언급만 합니다.

## 이 테마를 관통하는 한 문장

정규화(EWC)·재생(iCaRL)·증류(LwF)가 망각 방어의 세 갈래를 정초했고, 사전학습 시대에 prompt 기반(L2P)이 '백본을 안 건드린다'는 새 길을 열었으며, replay(CLEAR)는 RL 등에서 그 위상을 지킵니다. "옛것의 흔적을 무엇으로 지키나"의 계보이자, 사전학습이 판을 바꾼 이야기입니다.
