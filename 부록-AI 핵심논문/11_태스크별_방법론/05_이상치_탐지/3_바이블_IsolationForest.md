# 정상을 모형화하지 말고, 이상을 고립하라 — Isolation Forest

> **Isolation Forest** — Liu, Ting, Zhou, 2008, ICDM (DOI:10.1109/ICDM.2008.17)
> 원문: https://doi.org/10.1109/ICDM.2008.17 · 분류: 바이블

LOF[2]와 One-Class SVM[3]은 모두 "정상의 모습"(밀도, 영역)을 먼저 모형화했습니다. 둘 다 무겁고, 고차원·대용량에서 힘이 빠졌죠. 이 논문은 발상을 통째로 뒤집습니다. 정상을 모형화하지 말고, 이상을 직접 고립시키자는 것입니다. 이 발상의 전환으로 거리나 밀도 계산 없이 선형 시간에 동작해, 정형 데이터 이상탐지의 사실상 표준이 되었습니다.

## 배경: 정상을 모형화하는 비용

기존 방법의 공통점은 "정상의 모습"을 먼저 학습한다는 것입니다. LOF는 밀도를, One-Class SVM은 영역을 추정합니다. 그런데 이것은 비쌉니다. 밀도, 거리, 경계를 추정하려면 데이터 전체의 구조를 계산해야 하기 때문입니다. 또 한 가지 아쉬운 점은, 이상의 본질을 직접 쓰지 못한다는 것입니다. 이상은 "드물고(few) 다르다(different)"는 단순한 성질을 가지는데, 기존 방법은 이를 간접적으로만 이용했습니다.

> 핵심 질문: "이상은 적고 동떨어져 있다"는 성질만으로, 정상 모형 없이 직접 이상을 가려낼 수 없을까?

## 핵심 아이디어: 무작위로 쪼개 고립시키기

데이터를 무작위로 계속 쪼갭니다. 특성을 아무거나 하나 고르고, 그 범위 안에서 분할점도 무작위로 골라 둘로 나눕니다. 이걸 한 점이 혼자 남을 때까지 반복합니다. 핵심 관찰은 이것입니다. 이상치는 몇 번만 잘라도 혼자 떨어지고, 정상치는 무리 속이라 여러 번 잘라야 고립됩니다. 그래서 "고립까지 걸린 분할 횟수(경로 길이)"가 짧을수록 이상입니다.

> 비유 — 스무고개로 사람 찾기. 평범한 특징의 사람은 "안경 썼나요? 키가 큰가요?" 여러 질문이 필요합니다. 그런데 유독 특이한 사람(키 2m, 보라색 머리)은 한두 질문 만에 특정됩니다. 질문이 적게 드는 사람이 곧 이상치입니다.

## 어떻게 작동하나

```
① iTree(고립 트리): 데이터 부분표본을 무작위 특성·무작위 분할점으로 재귀 분할
   → 각 점이 잎에 닿기까지의 경로 길이를 잰다
② 숲(Forest): 이런 트리를 여럿 만들어 경로 길이를 평균낸다(랜덤성 안정화)
③ 이상 점수: s(x) ∝ 2^(−평균경로길이 / 정규화상수)
   경로가 짧을수록 점수가 1에 가까워(이상), 길수록 0.5 이하(정상)
④ 부분표본(subsampling): 작은 표본으로 트리를 지어, 대용량에서도 빠르고
   군집 가림(masking/swamping)을 줄인다
```

핵심은 정상을 전혀 모형화하지 않는다는 점입니다. 밀도도, 거리도, 경계도 계산하지 않습니다. 그냥 무작위로 쪼개고 경로 길이를 잽니다. 부분표본을 쓰는 것도 영리합니다. 전체가 아닌 작은 표본(예: 256개)으로 트리를 지으면 대용량에서도 빠르고, 이상치들이 서로를 가리는 현상(masking)이나 정상이 이상으로 오인되는 현상(swamping)을 줄입니다.

| 방법 | 모형화 대상 | 비용 |
|---|---|---|
| LOF | 밀도 | 무거움(이웃 탐색) |
| One-Class SVM | 영역(경계) | 무거움(커널) |
| Isolation Forest | 없음(직접 고립) | 선형 시간 |

## 임팩트: 왜 바이블이 되었나

- **정형 데이터 이상탐지의 사실상 표준**입니다. scikit-learn IsolationForest로 내장되어, 산업에서 1순위로 시도되는 방법입니다.
- **선형 시간과 낮은 메모리**입니다. 거리·밀도 계산이 없어 LOF·One-Class SVM보다 압도적으로 가볍고 잘 확장됩니다.
- **"이상을 직접 고립"이라는 패러다임**을 열었고, 이는 Deep Isolation Forest[4]로 딥 시대에 재해석됩니다.

## 한계와 주의점

- **축평행 분할의 한계**입니다. 특성 축에 평행한 분할만 하므로, 비스듬하거나 국소적인 이상 구조를 놓칠 수 있습니다. Deep iForest[4]가 신경망 표현으로 이를 보완합니다.
- **전역 성향**입니다. 전역적으로 드문 이상에 강하지만, 국소 이상(LOF가 잘 잡는)은 상대적으로 약할 수 있습니다.
- **고차원에서 무관 특성**이 많으면 무작위 분할의 효율이 떨어집니다.

## 한 줄 요약

> Isolation Forest는 정상을 모형화하는 대신 무작위 분할로 이상을 직접 고립시킨다. '적게 잘라도 혼자 떨어지면 이상'이라는 발상으로, 선형 시간으로 정형 이상탐지의 표준이 되었다.
