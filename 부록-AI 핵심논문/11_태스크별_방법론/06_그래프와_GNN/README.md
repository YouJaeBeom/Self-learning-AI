# 06 · 그래프와 GNN — 논문편 (바이블 → 최신)

노드와 엣지로 된 그래프에서 표현을 어떻게 배울까요. 소셜 네트워크, 분자, 지식그래프, 추천 시스템처럼 관계가 핵심인 데이터는 그래프로 다뤄야 합니다. 이 테마는 그 표현 학습이 랜덤워크에서 합성곱으로, 다시 어텐션과 트랜스포머로 진화한 계보를 따라갑니다. 연결만 보던 임베딩에서 특성을 녹이는 메시지 패싱으로, 다시 전역을 보는 트랜스포머로, 시야가 점점 넓어지는 이야기입니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | DeepWalk (2014) | 랜덤워크+Skip-gram 노드 임베딩의 효시 |
| 바이블 | node2vec (2016) | 편향 워크(p,q)로 이웃 구조를 조절 |
| 바이블 | GCN (2017) | 스펙트럴 합성곱 1차 근사 — 현대 GNN의 표준 |
| 바이블 | GraphSAGE (2017) | 이웃 샘플·집계로 귀납적 학습 |
| 최신 | GAT (2018) | 어텐션으로 이웃 중요도를 학습 |
| 최신 | GraphGPS (2022) | 위치인코딩+MPNN+전역 어텐션, 선형 그래프 트랜스포머 |

## 이 테마를 관통하는 한 문장

랜덤워크 임베딩(DeepWalk·node2vec)이 '연결의 표현'을 열었고, 합성곱(GCN)·샘플링 집계(GraphSAGE)가 특성+귀납을 더했으며, 어텐션(GAT)을 거쳐 그래프 트랜스포머(GraphGPS)로 확장됩니다. 연결만 보던 임베딩에서, 특성을 녹이는 메시지 패싱으로, 다시 전역을 보는 트랜스포머로 시야가 점점 넓어집니다.
