# 04 · 시계열 분석 (논문) 정리노트 — 계보와 요약

## 계보: 맞춤 학습에서 한 번 학습 후 어디든

1970년 Box-Jenkins가 ARIMA(AR+차분+MA)와 '식별-추정-진단' 절차로 시계열 예측을 방법론화했습니다. 반세기를 버틴 표준이었지만 선형이고 시계열마다 따로 학습한다는 한계가 있었죠. 2020년 DeepAR가 하나의 RNN을 여러 시계열로 함께 학습하고 분포까지 예측해(글로벌·확률) 딥러닝 예측을 산업에 안착시켰습니다. 같은 해 N-BEATS는 순환·어텐션 없이 FC 잔차 스택만으로 M4 SOTA와 해석 가능을 보였고, 2023년 PatchTST는 패치 토큰화와 채널 독립으로 트랜스포머의 장기 예측을 끌어올렸습니다. 2024년 Chronos(값을 토큰화해 T5로)와 TimesFM(패치를 디코더-온리로)이 사전학습 후 제로샷인 파운데이션 모델 시대를 열었습니다.

한 문장으로: **통계·선형·개별 모델(ARIMA) → 딥러닝·글로벌·확률(DeepAR) → 순수 구조·패치 트랜스포머(N-BEATS·PatchTST) → 사전학습 후 제로샷 파운데이션 모델(Chronos·TimesFM)로, '맞춤 학습'에서 '한 번 학습 후 어디든'으로 전환됐다.**

| 단계 | 논문 | 분류 | 계보적 역할 |
|---|---|:---:|---|
| 1 | Box-Jenkins / ARIMA (1970) | 바이블 | 식별-추정-진단으로 시계열 예측을 방법론화 (+ 선형·개별 한계) |
| 2 | DeepAR (2020) | 바이블 | 글로벌 공유 학습 + 확률적 예측으로 딥러닝 예측 안착 |
| 3 | N-BEATS (2020) | 최신 | FC 잔차 스택만으로 SOTA·해석 가능 — 단순함의 반격 |
| 4 | PatchTST (2023) | 최신 | 패치 토큰화 + 채널 독립으로 장기 예측 도약 (표준 부품화) |
| 5 | Chronos (2024) | 최신 | 값 양자화로 시계열을 '언어'로 → T5 제로샷 TS-FM |
| 6 | TimesFM (2024) | 최신 | 패치를 디코더-온리로 사전학습 → 공개 제로샷 TS-FM |

## 한 줄 요약 (논문별)

| 논문 | 분류 | 한 줄 핵심 |
|---|:---:|---|
| Box-Jenkins / ARIMA (Box & Jenkins, 1970) | 바이블 | AR+차분+MA, 식별→추정→진단을 잔차가 무작위 될 때까지 반복 |
| DeepAR (Salinas et al., 2020) | 바이블 | 하나의 RNN을 여러 시계열로 함께, 값 대신 분포를 출력 |
| N-BEATS (Oreshkin et al., 2020) | 최신 | 순환·어텐션 없이 FC 잔차 스택만으로 SOTA·성분 해석 |
| PatchTST (Nie et al., 2023) | 최신 | 시계열을 패치로 토큰화, 변수는 채널 독립 — 장기 예측 강화 |
| Chronos (Ansari et al., 2024) | 최신 | 실숫값을 정수 토큰으로 양자화해 T5로 학습, 제로샷 |
| TimesFM (Das et al., 2024) | 최신 | 패치를 양자화 없이 디코더-온리로 사전학습, 제로샷 |

보조 언급(별도 글 없음): Moirai(Woo et al., 2024, ICML Oral), Lag-Llama(Rasul et al., 2023) — 같은 TS-FM 흐름. DeepAR 저자 순서는 IJF 게재판 기준이고, ARIMA는 도서라 단일 DOI가 없습니다.

## 연결
