# 04 · 시계열 분석 (논문) — 바이블에서 파운데이션 모델까지

시간 순서가 있는 데이터로 미래를 예측합니다. 무엇이 그 예측을 가능케 할까요. 이 테마는 반세기 표준이던 통계 모델(ARIMA)에서, 딥러닝을 거쳐, 사전학습 후 제로샷인 시계열 파운데이션 모델까지 그 원전들을 따라갑니다. 핵심 전환은 "이 시계열에 맞춤 학습"에서 "한 번 사전학습 후 어디든"으로의 이동입니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | Box-Jenkins / ARIMA (1970) | 식별-추정-진단으로 ARIMA를 정초한 반세기 교과서 |
| 바이블 | DeepAR (2020) | 다수 시계열 공유 학습 + 확률적 예측의 산업 표준 |
| 최신 | N-BEATS (2020) | residual+FC 스택만으로 M4 SOTA, 해석 가능 |
| 최신 | PatchTST (2023) | 패치 토큰화 + 채널 독립으로 장기예측 도약 |
| 최신 | Chronos (2024) | 값을 토큰화해 T5 학습, 제로샷 TS-FM |
| 최신 | TimesFM (2024) | patched-decoder 디코더-온리 TS-FM |

## 이 테마를 관통하는 한 문장

ARIMA가 반세기 표준을 세웠고(1970), DeepAR가 딥러닝·확률 예측을 산업에 안착시켰으며(2020), N-BEATS·PatchTST가 순수 신경망·패치 트랜스포머로 성능을 끌어올린 뒤, Chronos·TimesFM이 '사전학습 후 제로샷'인 파운데이션 모델 시대를 열었습니다. "이 시계열에 맞춤 학습"에서 "한 번 사전학습 후 어디든"으로의 전환입니다.
