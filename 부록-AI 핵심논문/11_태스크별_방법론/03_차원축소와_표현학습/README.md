# 03 · 차원축소와 표현학습 — 바이블에서 최신까지

고차원 데이터를 적은 차원으로 옮기되, 무엇을 살릴 것인가? 그리고 더 나아가, 좋은 표현 자체를 레이블 없이 배울 수 있는가? 이 테마는 "고차원을 눈으로 보기"(t-SNE·UMAP)에서 "고차원의 좋은 표현을 스스로 배우기"(SimCLR·MAE)로, 차원축소가 표현학습으로 진화한 큰 줄기를 원전으로 잇습니다. 주어진 표현을 줄여 보는 것에서 좋은 표현을 배우는 것으로의 전환입니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | t-SNE (2008) | Student-t 무거운 꼬리로 군집을 또렷이 — 비선형 시각화의 표준 |
| 바이블 | UMAP (2018) | 위상·그래프로 전역 구조·속도 개선, 새 점 투영 가능 |
| 최신 | SimCLR (2020) | 증강 + projection head + 대형 배치 대조 자기지도 |
| 최신 | MAE (2022) | 75% 마스킹 + 비대칭 인코더-디코더 생성형 자기지도 |

PCA는 단일 논문이 아니라 고전 통계 기법(Pearson 1901 / Hotelling 1933)이라 별도 글로 다루지 않습니다(기초·심화편에서 직관으로 소개).

## 이 테마를 관통하는 한 문장

t-SNE(2008)와 UMAP(2018)이 고차원을 눈으로 보는 비선형 시각화를 표준화했고, 그 위에서 SimCLR(2020·대조)과 MAE(2022·마스킹)가 레이블 없이 좋은 표현 자체를 배우는 자기지도 표현학습으로 무게중심을 옮겼습니다. "주어진 표현을 줄여 보기"에서 "좋은 표현을 배우기"로의 전환입니다.
