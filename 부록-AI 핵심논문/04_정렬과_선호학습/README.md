# T04 · 정렬·선호학습 — 어떻게 사람 뜻에 맞추나

사전학습을 마친 모델은 다음 단어를 잘 맞힐 뿐, 도움이 되고 안전한 답을 하지는 않습니다. 이 둘 사이의 간극을 메우는 것이 정렬(alignment)입니다. 문제는 "좋은 답"을 규칙으로 적기 어렵다는 데 있습니다. 그래서 사람이나 AI의 선호를 모아 거기서 배우게 합니다. 이 테마는 그 정렬 기법이 사람 피드백에서 출발해 점점 단순해지고, 마침내 추론 능력까지 끌어올리는 데 이르는 흐름을 다섯 편의 논문으로 따라갑니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | InstructGPT (2022) | SFT → 보상 모델 → PPO, RLHF 3단계를 확립 |
| 바이블 | Constitutional AI (2022) | 헌법으로 모델이 스스로 비평·수정 (RLAIF) |
| 최신 | DPO (2023) | 보상 모델 없이 선호쌍을 직접 최적화 |
| 최신 | GRPO (2024) | 크리틱 없이 그룹 내 상대 보상으로 강화학습 |
| 최신 | DeepSeek-R1 (2025) | 정렬을 추론 인센티브로 확장, Nature 게재 |

## 이 테마를 관통하는 한 문장

사람 피드백 강화학습으로 정렬을 확립하고(InstructGPT), AI 피드백으로 확장하고(Constitutional AI), 보상 모델을 없애 단순화하고(DPO), 크리틱까지 없애 가볍게 하고(GRPO), 마침내 보상을 추론에 걸어 능력을 키웁니다(R1). "사람 뜻에 맞추기"가 점점 단순해지고 확장되는 이야기입니다. 부품을 하나씩 덜어 내면서도(보상 모델 → 크리틱) 더 강력해지는 과정을 보는 재미가 있습니다.
