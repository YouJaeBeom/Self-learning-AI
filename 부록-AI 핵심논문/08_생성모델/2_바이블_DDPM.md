# 노이즈 제거로 그림을 빚다 — DDPM

> **Denoising Diffusion Probabilistic Models** — Ho et al., 2020, NeurIPS (arXiv:2006.11239)
> 원문: https://arxiv.org/abs/2006.11239 · 분류: 바이블

GAN[2]은 사실적이었지만 불안정했습니다. 학습이 발산하기 쉽고 모드 붕괴에 시달렸죠. DDPM은 전혀 다른 발상으로 생성을 풉니다. 이미지에 노이즈를 조금씩 더했다가, 그 과정을 거꾸로 되돌리며 그림을 빚는 것입니다. 적대적 게임 대신 안정적인 회귀 문제로 생성을 바꿔, 고품질과 다양성을 함께 얻었습니다. 오늘날 Stable Diffusion, DALL-E 2, Imagen 같은 텍스트-이미지 생성의 폭발이 모두 이 DDPM 계보에서 나왔습니다.

## 배경: 안정적인 생성을 원하다

GAN의 두 가지 고질병은 학습 불안정성과 모드 붕괴였습니다. 두 네트워크의 미묘한 균형에 의존하다 보니 잘 학습시키기가 어려웠습니다. 연구자들은 GAN의 사실성을 유지하면서도, 더 안정적이고 다양성을 보존하는 생성 방법을 원했습니다. 이상적으로는 적대적 게임이 아니라, 명확한 목표를 향해 단조롭게 학습되는 지도학습 비슷한 문제로 생성을 바꾸고 싶었습니다.

> 핵심 질문: 순수한 노이즈에서 출발해, 점진적으로 다듬어 이미지를 만들 수 없을까?

## 핵심 아이디어: 더하고, 거꾸로 빼기

DDPM은 생성을 두 방향의 과정으로 봅니다. 전방 과정은 진짜 이미지에 노이즈를 조금씩 여러 단계에 걸쳐 더해 결국 완전한 잡음으로 만듭니다. 이 과정은 정해진 규칙이라 학습이 필요 없습니다. 역방 과정은 잡음에서 시작해 매 단계 조금씩 노이즈를 제거하며 이미지를 복원하는데, 바로 이 노이즈 제거를 신경망이 학습합니다.

> 비유 — 안개를 걷어 내는 조각. 형체 없는 안개(잡음)에서 출발해 한 겹씩 또렷하게 만들어 가면 마침내 형상이 드러납니다. 모델은 전체를 한 번에 그릴 필요 없이 "지금 낀 안개를 한 겹 걷는 법"만 배우면 됩니다. 어려운 문제를 수백 개의 쉬운 단계로 쪼갠 것입니다.

## 수식으로 들어가기

전방 과정은 분산 스케줄 `β_t`에 따라 각 단계에서 노이즈를 더합니다.

```
q(x_t | x_{t-1}) = N( x_t ; √(1−β_t)·x_{t-1} , β_t·I )
```

DDPM의 영리한 점은 여러 단계를 한 번에 건너뛸 수 있다는 것입니다. `α_t = 1−β_t`, `ᾱ_t = Π α_s`로 두면, 원본 `x_0`에서 임의의 시점 `x_t`를 닫힌 형태로 한 번에 샘플링할 수 있습니다.

```
x_t = √(ᾱ_t)·x_0 + √(1−ᾱ_t)·ε ,    ε ∼ N(0, I)
```

즉 `x_t`는 원본 이미지와 노이즈의 가중합입니다. 역방 과정에서 신경망 `ε_θ`는 "이 `x_t`에 섞인 노이즈 `ε`가 무엇이었나"를 예측하도록 학습됩니다. 변분 하한(ELBO)을 유도하면 복잡해 보이지만, 논문은 이것이 놀랍도록 단순한 손실로 정리됨을 보였습니다.

```
L_simple = E_{x_0, ε, t} [ ‖ ε − ε_θ(x_t, t) ‖² ]
```

이것은 그냥 "더해진 노이즈를 맞히는" 회귀 문제입니다. GAN의 minimax 게임과 달리 안정적인 단일 목표라 다루기 쉽습니다. 학습된 `ε_θ`로 잡음에서 시작해 노이즈를 단계적으로 제거하면 이미지가 생성됩니다.

```
전방(고정): 이미지 → +노이즈 → … → 순수 잡음
역방(학습): 순수 잡음 → -노이즈 → … → 이미지   (ε_θ가 노이즈를 예측해 제거)
```

| 항목 | GAN | DDPM |
|---|---|---|
| 학습 방식 | 적대적 게임 | 노이즈 회귀(안정) |
| 다양성 | 모드 붕괴 위험 | 잘 보존 |
| 생성 속도 | 한 번에 빠름 | 여러 단계로 느림 |

참고로 이 노이즈 예측은 데이터 분포의 기울기(score)를 추정하는 관점과 연결되어, Score-SDE[3] 같은 보조 계보와도 이어집니다.

## 임팩트: 왜 바이블이 되었나

- **현대 확산 모델의 토대**입니다. Stable Diffusion, DALL-E 2, Imagen 등 텍스트-이미지 생성의 폭발이 모두 DDPM 계보에서 나왔습니다.
- 안정적이고 고품질인 생성 경로를 제시해, 생성의 주류를 GAN에서 확산으로 옮기는 전환점이 되었습니다. 적대적 게임의 어려움을 회귀 문제의 단순함으로 대체한 것입니다.
- 이후 잠재공간 확산(SD), 트랜스포머 백본(DiT), 직선 경로(SD3)로 빠르게 발전합니다. 이 테마의 나머지가 모두 그 후손입니다.

## 한계와 주의점

- **샘플링이 느립니다.** 여러 단계의 반복 노이즈 제거가 필요해, 한 장을 만드는 데 GAN보다 오래 걸립니다. 원논문은 1,000단계를 썼습니다(후속 연구가 단계 수를 크게 줄임).
- 픽셀 공간에서 직접 돌리면 연산과 메모리가 큽니다. 고해상도일수록 비용이 폭발하는데, 이것이 다음 글 잠재공간 확산[4]의 핵심 동기입니다.

## 더 읽을거리와 연결

- 이 책 내부: 앞 글 **GAN**[2](불안정성), 다음 글 **Stable Diffusion**[4](느림·비용을 잠재공간으로 해결).
- 심화편 S2(전방·역방·score 관점)와 함께 보면 좋습니다.

## 한 줄 요약

> DDPM은 노이즈를 점진적으로 더했다가 거꾸로 제거하는 확산으로, GAN보다 안정적이고 고품질인 생성을 열어 현대 확산 모델의 토대가 되었다. 대신 샘플링이 느리다.
