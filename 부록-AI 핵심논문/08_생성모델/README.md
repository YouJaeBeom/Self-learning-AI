# T08 · 생성모델 — 분포를 배워 새것을 짓다

분류가 주어진 것을 가르는 일이라면, 생성은 없던 것을 만드는 일입니다. 분류기는 "이게 고양이냐 개냐"를 답하지만, 생성기는 한 번도 존재하지 않은 새 고양이 사진을 그려야 합니다. 이를 위해서는 데이터 분포 자체를 배워야 합니다. 이 테마는 그 분포를 배워 새 이미지를 짓는 계보를 따라갑니다. 적대학습(GAN)과 확산(DDPM)이라는 두 바이블에서 출발해, 잠재공간으로 비용을 낮추고, 트랜스포머로 확장성을 더하고, 직선 경로로 효율을 끌어올리는 최신 텍스트-이미지 SOTA까지 이어집니다.

## 바이블에서 최신까지, 한눈에

| 분류 | 논문 | 핵심 한 줄 |
|---|---|---|
| 바이블 | GAN (Goodfellow et al., 2014) | 생성자 대 판별자, 적대적 minimax |
| 바이블 | DDPM (Ho et al., 2020) | 노이즈를 점진적으로 제거하는 확산 |
| 최신 | Latent Diffusion / SD (Rombach et al., 2021/22) | 잠재공간에서 확산 → 연산 절감·대중화 |
| 최신 | DiT (Peebles & Xie, 2022/23) | 확산 백본을 U-Net에서 트랜스포머로 |
| 최신 | SD3 / Rectified Flow (Esser et al., 2024) | rectified flow + MM-DiT로 최신 SOTA |

## 이 테마를 관통하는 한 문장

생성 모델의 큰 줄기는 안정성과 효율을 좇아 흘러왔습니다. GAN이 적대학습으로 패러다임을 열었지만 불안정했고, DDPM이 안정적 고품질로 주류를 이어받았으며, 잠재공간(SD)·트랜스포머 백본(DiT)·직선 경로(SD3)로 비용과 확장성을 차례로 개선했습니다. "어떻게 더 안정적으로, 더 싸게, 더 잘 생성하나"의 이야기입니다.
