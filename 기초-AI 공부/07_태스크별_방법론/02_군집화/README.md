# 02 · 군집화 (Clustering) — 정답 없이 묶기

> **이 태스크의 목적**
> 정답표(레이블) 없이, 데이터 안에 숨은 *그룹*을 스스로 찾아내는 방법을 잡는다.
> "비슷한 것끼리 묶는다"는 한 문장이 어떻게 알고리즘이 되는지를 직관으로 이해한다.

> ← **연결** — [기초편 2단계 · 학습의 종류](../../02_머신러닝_개념과_방법론/1_학습의_종류.md)에서 *지도학습 vs 비지도학습*을 잡고 오면 좋습니다. 군집화는 **비지도학습의 대표 주자**입니다.

---

## 이 태스크에서 답하는 질문

이 폴더를 다 읽고 나면, 다음을 **말로** 답할 수 있어야 합니다.

- "정답이 없다"는 군집화는 분류와 무엇이 다른가?
- k-means는 *어떻게* 비슷한 것끼리 묶는가? (중심 잡고 → 할당 → 갱신, 이 반복)
- "비슷하다"를 컴퓨터가 재려면 무엇이 필요한가? (거리)
- 정답이 없는데, 잘 묶었는지 *어떻게* 채점하는가? (실루엣)

> 한마디로 이 폴더의 핵심 질문 — **"정답표가 없는데, 무엇을 보고 '잘 묶었다'고 말할 수 있을까?"**
