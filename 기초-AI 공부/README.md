## 이 책은 무엇인가

원본 커리큘럼은 "무엇을 공부할지"를 알려주는 **지도**입니다.
이 책은 그 지도를 따라 실제로 걸어갈 수 있도록 **길 위의 풍경을 말로 풀어 쓴 본문**입니다.

- 커리큘럼이 *목차*라면, 이 책은 *각 장의 내용*입니다.
- 외부 강의·교재로 넘어가기 전에, **개념의 직관과 큰 그림을 먼저 잡는 용도**입니다.
- 모든 설명은 〈목적 → 개념 → 방법론 → 수학〉 네 렌즈를 따릅니다.

> **이 책의 한계** — 이 책은 개념의 *직관과 연결*을 잡아 주는 안내서입니다.
> 깊은 증명·풍부한 예제·연습문제는 각 장이 안내하는 외부 자료(3Blue1Brown, MML-book, ISL 등)에서 채워야 합니다.
> 이 책으로 길을 잡고, 외부 자료로 깊이를 더하세요.

## 책을 읽는 법

각 단계 폴더는 하나의 **장(章)**입니다. 장 안의 파일은 순서대로 읽도록 번호가 붙어 있습니다.

1. **장의 `README.md`를 먼저 읽으세요.** 그 장의 목적·지도·읽는 순서가 담겨 있습니다.
2. **본문 파일을 번호 순서대로 읽으세요.** 각 파일은 한 번에 읽을 수 있는 분량(한 개념)입니다.
3. **장의 마지막 `정리노트.md`로 마무리하세요.** 〈한 줄 요약 → 핵심 수식 → 이해 점검〉으로 스스로 점검합니다.
4. **막히면 자료를 바꾸세요.** 각 본문 끝의 "더 깊이 파기" 링크로 영상·교재·그림을 병행합니다.

### 본문에 자주 나오는 표시

| 표시 | 뜻 |
|---|---|
| **목적** | 이 개념은 *어떤 문제*를 풀려고 나왔는가 (동기·직관) |
| **핵심 개념** | 반드시 붙잡아야 할 용어와 아이디어 |

| **이해 점검** | 말로 설명할 수 있으면 통과 |

## 학습 원칙 4가지 (커리큘럼에서 그대로)

1. **설명할 수 있어야 안다** — 비전공자에게 말로 설명할 수 있을 때 통과.
2. **수식은 의미부터** — 기호를 읽기 전에 "이 식이 무엇을 최소화/최대화하려는가"를 먼저 묻는다.
3. **한 줄 요약 노트** — 주제마다 〈목적 / 핵심 아이디어 / 핵심 수식〉 세 줄을 직접 적는다.
4. **막히면 자료를 바꾼다** — 같은 개념을 영상·교재·그림 세 형태로.
