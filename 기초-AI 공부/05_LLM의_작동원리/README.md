# 5단계 · LLM의 작동 원리

> **이 장의 목적**
> "GPT 같은 거대 언어모델은 *대체 무엇을 학습한 것인가*"에 답합니다.
> 다음 토큰 예측·토큰화·사전학습 규모·스케일링·샘플링의 원리를 이해합니다.

---

## 이 장에서 답하는 질문

이 장을 다 읽고 나면, 다음 질문에 **말로** 답할 수 있어야 합니다.

- **'다음 토큰 예측'**이라는 단순한 목표가 어떻게 광범위한 언어 능력으로 이어지는가?
- 토큰화(BPE)는 왜 필요하고, 컨텍스트 윈도우란 무엇인가?
- 사전학습의 '규모'(데이터·파라미터·연산)는 서로 어떤 관계인가? 사전학습→파인튜닝→정렬은 무엇인가?
- **스케일링 법칙**과 **창발(emergence)**은 무엇인가? 크게 만들면 왜 더 똑똑해지는가?
- 퍼플렉서티는 무엇을 재는가? temperature·top-k·top-p는 출력을 어떻게 바꾸는가?

> 한마디로 이 장의 핵심 질문은 이것입니다 — **"왜 '다음 단어 맞히기'라는 단순한 목표가, 이토록 강력한가?"**

## 이 단계에 필요한 수학

새 수학은 적습니다. **1단계의 확률·통계가 LLM에서 어떻게 쓰이는지**를 봅니다.

- 언어모델의 목표함수 = 토큰열의 **로그우도 최대화** (= **교차엔트로피 최소화**). → 1단계 ⑤·확률·통계의 직접 응용.
- **퍼플렉서티(perplexity)**의 정의와 의미 (교차엔트로피와 한 몸).
- 스케일링 법칙의 **멱법칙(power-law)** 형태(개념 수준).
- 샘플링: temperature·top-k·top-p가 **확률분포를 어떻게 조절**하는가.

> 4단계 트랜스포머가 '모양'이었다면, 5단계는 그 모양에 **무엇을, 어떻게 학습시키는가**입니다.
