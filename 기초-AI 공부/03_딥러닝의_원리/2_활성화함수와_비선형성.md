# 2. 활성화 함수와 비선형성 — 왜 '구부림'이 필수인가

앞 글에서 한 층은 `z = Wx + b` 다음 `a = f(z)`라고 했습니다. 여기서 `f`가 **활성화 함수(activation function)**입니다. 가중합 뒤에 그냥 한 번 거치는 양념처럼 보이지만, 사실 이 `f` 하나가 없으면 **신경망을 깊게 쌓는 의미가 통째로 사라집니다.**

이 글은 그 이유를 수식으로 증명하고, 대표 활성화 함수 세 가지(ReLU·시그모이드·tanh)를 직관·기울기·쓰임새까지 비교한 뒤, "어떤 함수를 어디에 쓰는가"라는 실전 감각까지 잡습니다. 결론을 먼저 던지면 이렇습니다.

> 선형변환이 '펴고 늘이기'라면, 활성화 함수는 '접기'다. 접기가 있어야 깊이가 표현력이 된다.

---

## 목적 — 층을 쌓는 '의미'를 지키기

질문 하나로 시작합니다. **"활성화 함수를 빼고 `Wx + b`만 여러 층 쌓으면 어떻게 될까?"**

답은 충격적입니다. **아무리 많이 쌓아도, 결국 단 한 층과 똑같아집니다.** 100층을 쌓든 1000층을 쌓든, 활성화가 없으면 표현력은 선형회귀(직선 하나) 수준에 머뭅니다. 깊이가 통째로 헛수고가 되는 거죠.

활성화 함수는 바로 이 붕괴를 막는 장치입니다. 즉 활성화 함수의 목적은 **"깊이가 진짜 힘을 갖게 만드는 것"**입니다. 1번 글의 결재 라인 비유로 말하면, 각 직급이 *자기만의 판단*을 더해야 결재 단계가 의미를 갖는다는 그 통찰이 여기서 수식으로 확인됩니다.

---

## 핵심 개념 — 비선형성과 대표 함수들

### ① 왜 비선형성(non-linearity)인가 — 핵심 논리

핵심 논리는 한 줄입니다. **선형변환을 아무리 합성해도 선형변환 하나일 뿐**이라는 것.

```
W₂(W₁x + b₁) + b₂ = (W₂W₁)x + (W₂b₁ + b₂)
```

괄호 안의 `W₂W₁`을 새 행렬 `W′`로, `W₂b₁ + b₂`를 새 벡터 `b′`로 묶으면 그냥 `W′x + b′` — 한 층입니다. 두 개의 선형층이 정확히 하나로 접혀 버립니다.

- 활성화 없이 쌓는 것은 **여러 장의 투명 필름을 겹쳐도 결국 한 장인** 것과 같습니다. 투명 필름을 100장 겹쳐도 빛은 직선으로 통과하죠.
- **중간에 비선형 `f`를 끼워 넣으면**, 더 이상 하나로 합쳐지지 않습니다. `f(W₁x + b₁)`는 행렬 곱으로 풀 수 없으니까요. 층마다 공간을 *구부려*, 직선으로는 못 긋던 복잡한 경계를 만들 수 있게 됩니다.

> 한 문장: **비선형성이 있어야 '깊이'가 표현력으로 바뀐다.** 이것이 1번 글에서 본 보편 근사 정리의 전제 조건이기도 합니다 — 정리는 "비선형 활성화가 있을 때" 성립합니다.

### ② 대표 활성화 함수 셋 — 식·모양·기울기

세 함수를 식과 함께 봅니다. 모양과 기울기를 함께 보는 게 핵심입니다.

- **ReLU(Rectified Linear Unit)**: `f(z) = max(0, z)`. 음수는 0으로 자르고 양수는 그대로 통과시킵니다. 현대 신경망의 사실상 기본값입니다.
  - 미분: `z > 0`이면 기울기 1, `z < 0`이면 기울기 0. **양수 구간에서 기울기가 항상 1**이라 깊은 신경망에서도 기울기가 잘 흐릅니다.
  - 장점: 계산이 극도로 쌉니다(비교 한 번). 그리고 기울기 소실에 강합니다(6번 글과 연결).
  - 약점: 음수 입력에서 기울기가 0이라, 어떤 뉴런이 계속 음수만 받으면 영영 안 깨어나는 **죽은 ReLU(dying ReLU)** 문제가 생깁니다. 이를 보완한 변형이 Leaky ReLU(`f(z)=max(0.01z, z)`), GELU(트랜스포머에서 표준) 등입니다.
- **시그모이드(sigmoid)**: `f(z) = 1 / (1 + e⁻ᶻ)`. 어떤 값이든 **0~1 사이**로 부드럽게 눌러 줍니다.
  - 미분: `f′(z) = f(z)(1 − f(z))`. 최댓값이 `z=0`에서 0.25, 양 끝(±5쯤)에서는 거의 0입니다.
  - 쓰임: 확률처럼 0~1이 필요할 때(이진 분류의 마지막 층). 단점: 양 끝이 평평해 기울기가 거의 0이 되어 **기울기 소실**의 대표적 원흉입니다.
- **하이퍼볼릭 탄젠트(tanh)**: `f(z) = (eᶻ − e⁻ᶻ)/(eᶻ + e⁻ᶻ)`, 값을 **−1~1 사이**로 누릅니다. 시그모이드를 위아래로 늘려 **0 중심**으로 옮긴 형태입니다.
  - 미분: `f′(z) = 1 − f(z)²`, 최댓값이 `z=0`에서 1입니다(시그모이드의 4배).
  - 0 중심이라 시그모이드보다 학습이 잘 되는 편이지만, 양 끝이 평평한 문제는 그대로 남습니다.

세 함수를 한눈에 비교하면 이렇습니다.

| 함수 | 식 | 출력 범위 | 최대 기울기 | 0 중심 | 주된 쓰임 | 약점 |
|---|---|:---:|:---:|:---:|---|---|
| **ReLU** | `max(0, z)` | `[0, ∞)` | 1 (양수 구간) | 아니오 | 은닉층 기본값 | 죽은 ReLU |
| **시그모이드** | `1/(1+e⁻ᶻ)` | `(0, 1)` | 0.25 | 아니오 | 이진 분류 출력 | 기울기 소실 |
| **tanh** | `(eᶻ−e⁻ᶻ)/(eᶻ+e⁻ᶻ)` | `(−1, 1)` | 1 | 예 | RNN 은닉 상태 | 기울기 소실 |

기울기의 모양을 그림으로 보면 ReLU가 왜 깊은 신경망에 유리한지 단번에 보입니다.

```
시그모이드의 기울기              ReLU의 기울기
  0.25 ┤    ╭─╮                  1.0 ┤        ┌────────
       │   ╱   ╲                     │        │
       │  ╱     ╲                    │        │
   0.0 ┤─╯       ╰─              0.0 ┤────────┘
       └──────────              z    └──────────  z
   끝으로 갈수록 0 → 소실        양수면 늘 1 → 잘 흐름
```

### ③ 어디에 쓰나 — 은닉층 vs 출력층

활성화 함수 선택은 "은닉층이냐 출력층이냐"로 크게 갈립니다.

- **은닉층(중간)**: 보통 **ReLU**(또는 GELU·SiLU 같은 변형). 빠르고 기울기가 잘 흐르기 때문입니다.
- **출력층(마지막)**: 문제의 종류에 맞춰 고릅니다.
  - 이진 분류 → **시그모이드**(0~1 확률 하나).
  - 다중 분류 → **softmax**(여러 후보에 합이 1인 확률을 배분, 1단계의 범주형 분포·3번 글의 어텐션과 연결).
  - 회귀(연속값 예측) → **활성화 없이** 그대로 출력(값을 0~1로 누르면 안 되니까).

> 기억할 한 줄: **은닉층은 '잘 흐르게'(ReLU), 출력층은 '문제에 맞게'(시그모이드/softmax/없음).** 은닉층은 학습이 목적, 출력층은 답의 형태가 목적이라 기준이 다릅니다.

## 직관 — 종이접기로 경계 만들기

평평한 종이 위에 빨간 점과 파란 점이 뒤섞여 있는데, **직선 한 번으로는** 둘을 못 가른다고 합시다.

- `Wx + b`(선형)만으로는 **직선밖에** 못 긋습니다. 아무리 여러 번 해도 '직선의 직선'은 직선입니다.
- 그런데 활성화 함수로 종이를 **한 번 접으면(구부리면)**, 접힌 종이 위에서 그은 직선은 펼친 종이에선 *꺾인 선*이 됩니다. ReLU가 정확히 이 '접기'를 합니다 — `z=0`을 경계로 종이를 꺾는 거죠.
- 층을 쌓으며 **접고 또 접으면**, 아무리 뒤엉킨 빨강·파랑이라도 가르는 복잡한 경계를 만들 수 있습니다.

```
선형만 (펴기만):              활성화 추가 (펴기+접기):
   빨강 ●  ● 파랑                빨강 ●╲  ╱● 파랑
   ─────────────  직선뿐           ●  ╲╱  ●
   파랑 ●  ● 빨강              ╱╲ 꺾인 경계로 분리 가능
   → 못 가름                     → 가름
```

> 활성화 함수 = **공간을 접는 손**. 선형변환이 '펴고 늘이기'라면, 활성화는 '접기'입니다. 접기가 있어야 복잡한 모양이 나옵니다. 깊은 신경망이 강력한 건 "수많은 접기를 합성"하기 때문입니다.

---

## 방법론 — 한 층의 두 동작

```
       선형변환(펴기)        비선형(접기)
입력 ──▶ z = Wx + b ──▶ a = f(z) ──▶ 다음 층
        공간을 늘이고 섞음    공간을 구부림
```

- 모든 층은 이 **'펴기 → 접기'** 두 동작의 짝입니다.
- 펴기만 반복(활성화 없음) → 결국 한 번 편 것과 같음(앞에서 본 붕괴).
- 펴기·접기를 번갈아 쌓음 → 임의의 복잡한 함수에 가까워짐(보편 근사, 1번 글).

여기에 한 가지 실용적 조건이 붙습니다. 활성화 함수는 **거의 모든 곳에서 미분 가능**해야 합니다. 3번 글에서 보듯 학습은 기울기(미분)를 거슬러 흘려보내는 일인데, 미분이 정의되지 않으면 그 통로가 막히기 때문입니다. ReLU는 `z=0`에서 미분이 정의되지 않지만, 그 한 점은 실전에서 0 또는 1로 임의 처리해도 문제없습니다("거의 모든 곳에서" 미분 가능하면 충분).

## 수학 — 붕괴를 보여 주는 식

먼저, 활성화가 없으면 왜 무너지는지 다시 식으로 못 박습니다.

```
활성화 없음:  W₂(W₁x + b₁) + b₂ = (W₂W₁)x + (W₂b₁ + b₂) = W′x + b′
```

> **이 식의 의미:** 활성화 없이 선형층을 두 번 쌓아도, 괄호를 풀면 새 가중치 `W′`·`b′` 하나로 묶인다. → **두 층이 한 층과 똑같다.** 깊이가 무의미해진다는 증명이다. 같은 논리로 100층도 한 층으로 접힌다.

다음으로, 비선형이 끼면 이 붕괴가 막힌다는 것을 봅니다.

```
ReLU:    f(z) = max(0, z),       f′(z) = 1 (z>0),  0 (z<0)
sigmoid: f(z) = 1/(1 + e⁻ᶻ),     f′(z) = f(z)(1 − f(z))
tanh:    f(z) = tanh(z),         f′(z) = 1 − f(z)²
```

> **이 식의 의미:** ReLU는 음수를 0으로 '꺾어' 비선형을 만든다(접기). 시그모이드·tanh는 값을 부드럽게 눌러 확률·정규화에 쓴다. `f(W₁x + b₁)`처럼 비선형이 끼면 행렬 곱으로 풀어 묶을 수 없으니 위의 붕괴가 일어나지 않는다.

미분 식을 나란히 두면 6번 글의 기울기 소실이 미리 보입니다. 시그모이드·tanh는 양 끝에서 `f′ ≈ 0`이라, 역전파 때 이 값을 층마다 곱하면 기울기가 0으로 사그라듭니다. 반면 ReLU는 양수 구간에서 `f′ = 1`이라 곱해도 줄지 않습니다. **활성화 함수의 선택이 곧 기울기가 살아남느냐의 문제**라는 점이 여기서 드러납니다.

---

## 흔한 오해

- **"활성화 함수는 부수적 장식이다"가 아닙니다.** 활성화 함수는 신경망의 **존재 이유**에 가깝습니다. 빼면 아무리 깊은 신경망도 선형회귀로 쪼그라듭니다. 양념이 아니라 골격입니다.
- **"시그모이드가 직관적이니 은닉층에도 좋다"가 아닙니다.** 시그모이드는 양 끝이 평평해 기울기가 0에 가까워지고, 깊은 신경망에서 **기울기 소실**을 일으킵니다. 그래서 은닉층은 보통 ReLU를 씁니다(시그모이드는 출력층에만).
- **"비선형이면 아무 함수나 된다"가 아닙니다.** 거의 모든 곳에서 미분 가능해야 역전파로 기울기를 흘릴 수 있습니다. 또 너무 평평하거나 너무 가파르면 학습이 망가집니다. '학습이 잘 흐르는가'라는 조건이 늘 붙습니다.
- **"ReLU는 완벽하다"가 아닙니다.** 음수 입력에서 기울기가 0이라 일부 뉴런이 죽는 문제가 있습니다. 그래서 Leaky ReLU·GELU·SiLU 같은 변형이 나왔고, 특히 트랜스포머는 GELU를 주로 씁니다.

---

## 한 줄 요약

> **활성화 함수(비선형)가 없으면, 층을 아무리 쌓아도 한 층과 같다.**
> 선형변환은 '펴기', 활성화는 '접기'. 접기가 있어야 깊이가 표현력이 된다. 은닉층은 보통 ReLU(기울기 잘 흐름), 출력층은 문제에 맞춰(이진=시그모이드, 다중=softmax, 회귀=없음). 활성화의 기울기 모양이 곧 기울기 소실(6번 글)을 좌우한다.
