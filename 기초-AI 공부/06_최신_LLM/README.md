# 6단계 · 최신 LLM — 정렬 · 추론 · 효율 · 응용

> **이 장의 목적**
> 2025~2026년 최전선의 개념(정렬·추론 모델·MoE·RAG·에이전트)이
> *각각 어떤 문제를 풀기 위한 것인지*를 이해한다. 수학은 모두 **개념 수준**.

## 이 장에서 답하는 질문

이 장을 다 읽고 나면, 다음 질문에 **말로** 답할 수 있어야 합니다.

- RLHF와 DPO는 '무엇을 위한' 단계이고, 서로 어떻게 다른가?
- 추론 모델(o-시리즈·DeepSeek-R1)은 일반 LLM과 무엇이 다른가?
- MoE(전문가 혼합)는 왜 효율적인가?
- RAG는 환각(hallucination)을 어떻게 줄이는가?
- 지금의 LLM이 가진 대표적 한계 3가지는?

> 한마디로 이 장의 핵심 질문은 이것입니다 — **"똑똑해진 LLM을, 어떻게 *원하는 대로·싸게·믿을 수 있게* 만드는가?"**

## 이 단계에 필요한 수학

모두 **개념 수준**입니다. 식의 형태보다 "무엇을 하려는 아이디어인가"에 집중하세요.

- 강화학습의 **보상·정책** 개념, RLHF의 목적함수 직관
- **DPO**가 보상 모델 없이 선호도를 직접 최적화하는 아이디어
- **MoE**의 게이팅(gating)과 희소 활성화(sparse activation)
- 임베딩 유사도(**코사인 유사도**)와 벡터 검색의 원리

> 4단계의 〈임베딩=의미 벡터, 내적=유사도〉가 RAG에서, 1단계의 〈최대화/최소화〉 감각이 정렬에서 다시 쓰입니다.
