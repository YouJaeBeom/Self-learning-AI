# 2단계 · 머신러닝의 개념과 방법론

> **이 장의 목적**
> 딥러닝에 들어가기 전에, **모든 학습 모델이 공유하는 공통 원리**(일반화·과적합·평가)를 잡는다.
> 이것을 모르면 딥러닝이 잘 되는지 안 되는지조차 해석할 수 없다.

## 이 장에서 답하는 질문

이 장을 다 읽고 나면, 다음 질문에 **말로** 답할 수 있어야 합니다.

- 지도학습·비지도학습·강화학습은 각각 무엇이 다른가?
- 과적합(overfitting)은 왜 생기고, 정규화(regularization)는 그걸 어떻게 막는가?
- 편향–분산 트레이드오프(bias–variance tradeoff)란 무엇인가?
- 정밀도와 재현율은 어떻게 다르고, 언제 무엇이 더 중요한가?
- 로지스틱 회귀는 어떻게 '확률'을 출력하는가?

> 한마디로 이 장의 핵심 질문은 이것입니다 — **"모델이 '잘 배웠다'는 것을 어떻게 확인하는가?"**

## 이 단계에 필요한 수학

수식이 처음으로 본격 등장하지만, **모두 의미 수준**입니다. 외우지 말고 "무엇을 줄이려는 식인가"만 읽으세요.

- 선형회귀의 **최소제곱**과 정규방정식 (의미 수준)
- 로지스틱 회귀의 **시그모이드(sigmoid)** 와 로그우도(log-likelihood)
- 정규화 항(**L1·L2**)이 손실에 더해지는 *이유*
- 평가지표의 정의: 정확도·정밀도·재현율·F1·ROC-AUC

> 1단계(머신러닝 수학)의 〈경사하강법 `θ ← θ − η·∇L(θ)`〉과 〈교차엔트로피 `H(p,q) = −Σ p(x) log q(x)`〉가 여기서 처음으로 '쓰이는 모습'으로 다시 나타납니다.
