# S5 · 정렬·후학습 심화 — 베이스 모델을 대화 모델로

기초편 6단계 1절(RLHF·DPO)을 파이프라인 수준으로 확장하는 모듈입니다. "사전학습으로 똑똑해진 모델을, 실제로 어떤 단계를 거쳐 쓸모 있고 안전하게 만드는가?"에 답합니다. 기초편의 한 편을 다섯 편으로 풀어내, SFT부터 합성 데이터까지 후학습의 전 과정을 따라갑니다.

먼저 기초편 6단계 1절(정렬: RLHF와 DPO)을 읽고 오면 좋습니다. 이 모듈은 그 한 편을 다섯 편으로 펼친 것이라, 그 토대 위에서 각 단계가 한 층씩 깊어집니다.

## 이 모듈에서 답하는 질문

- 사전학습이 끝난 '베이스 모델'을 대화 모델로 바꾸는 후학습(post-training)의 전체 순서는
- 보상 모델(reward model)은 무엇을 학습하고, RLHF의 목적함수 `보상 − KL`은 무슨 뜻인가
- DPO는 왜 보상 모델을 없앨 수 있었고, GRPO·KTO·IPO는 각각 무엇을 바꿨나
- 사람 대신 AI가 피드백을 주는 RLAIF·Constitutional AI는 무엇을 푸는가
- 후학습용 합성 데이터는 왜 쓰이고, 무엇이 위험한가

한마디로 "좋은 답을 규칙으로 못 적는다면, 어떤 파이프라인으로 모델에 가르칠 것인가?"입니다.

## 이 모듈에 필요한 수학 (모두 의미 수준)

RLHF 목적함수 `보상 − β·KL(현재 정책 ‖ 기준 정책)`의 의미, DPO의 암묵적 보상(implicit reward) 재매개화 아이디어(유도는 원문으로), 그리고 기초편 수준의 시그모이드·로그우도면 충분합니다. 유도와 증명은 넣지 않습니다. "이 식이 무엇을 최대화하거나 최소화하려는가"만 말로 잡습니다.
