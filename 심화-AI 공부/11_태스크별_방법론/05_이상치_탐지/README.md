# 05 · 이상치 탐지 (심화) — 다섯 갈래의 지도와 평가의 난점

기초편의 "정상에서 벗어남"이라는 한 직관이, 실제로는 여러 갈래의 방법론으로 갈라진다는 걸 잡는 모듈입니다. 밀도·고립·경계·딥러닝·시계열 — 각 갈래가 어떤 가정 위에서 동작하고, 어디서 무너지는지를 비교합니다. 그리고 이상치 탐지에서 모델만큼이나 어려운 평가 문제를 정직하게 다룹니다.

기초편 05 트랙에서 이상 점수·z-score·거리·PR-AUC의 직관을 잡고 오면 좋습니다. 새 모델 수학은 깊이 다루지 않습니다. 여기서는 각 갈래의 가정과 트레이드오프가 주인공입니다.

## 이 모듈에서 답하는 질문

- "정상의 모습"을 잡는 방법은 왜 한 가지가 아닌가, 밀도·고립·경계·재구성은 무엇을 정상으로 가정하나
- 단순 거리의 다밀도 약점을 LOF는 어떻게 푸나, iForest는 왜 거리를 아예 안 쓰나
- 딥러닝(오토인코더·Deep SVDD)은 고차원·이미지에서 무엇을 더하나
- 시계열 이상은 왜 별도 갈래인가
- 평가가 왜 이렇게 까다로운가, ROC는 왜 후하게 보이고 PR-AUC를 믿는가

## 이 모듈에 필요한 수학 (의미 수준)

국소 밀도와 상대 밀도비(LOF의 핵심), 트리에서 한 점을 고립시키는 데 필요한 평균 분할 횟수(iForest의 경로 길이), 잠재 공간의 초구 반경 최소화(Deep SVDD), 재구성 오차를 이상 점수로 쓰기(오토인코더), PR-AUC와 ROC-AUC의 차이와 불균형에서의 행동이면 충분합니다. 여기서는 각 갈래의 가정과 트레이드오프가 주인공입니다.
