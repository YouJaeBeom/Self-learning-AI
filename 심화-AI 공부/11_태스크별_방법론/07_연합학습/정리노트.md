# 07 연합학습 (심화) · 정리노트

이 모듈의 본문을 한 장으로 압축했습니다. 두 글의 핵심을 다시 짚고, 도전과 해법의 지도, 대표 논문을 정리합니다.

## 두 글의 요지

| 글 | 핵심 한 줄 |
|---|---|
| 1. 세 도전 | non-IID(client drift)·통신 비용·개인화 — FedProx(묶기)·압축/PEFT·Ditto/FedBN로 푼다 |
| 2. 보안 집계 & 연합 LLM | 상쇄 마스크로 개별 업데이트를 가리고(보안 집계), PEFT로 거대 LLM을 연합 파인튜닝 |

## 도전과 해법의 지도

| 도전 | 해법 | 핵심 |
|---|---|---|
| non-IID / drift | FedProx | proximal term으로 글로벌에 묶기 |
| 통신 비용 | 압축·부분참여·PEFT | 보낼 양·횟수 줄이기 |
| 개인화(공정·견고) | Ditto | 정규화로 글로벌↔로컬 절충 |
| feature shift | FedBN | BN 레이어 로컬 유지 |
| 서버 신뢰 부족 | Secure Aggregation | 상쇄 마스크로 합계만 노출 |
| 거대 LLM | FederatedScope-LLM | PEFT 통합 + 벤치마크 |

## 한 장으로 보는 핵심

모든 개인화 해법의 본질은 로컬의 자유와 글로벌의 공통성 사이 균형 조절입니다(μ, λ, 무엇을 공유하고 로컬로 둘지). 연합학습의 진짜 병목은 계산이 아니라 통신이므로 압축·PEFT가 핵심입니다. 프라이버시는 연합(원본 안 보냄)에서 보안 집계(개별 못 봄)로, 차등 프라이버시(흔적 흐림)로 층층이 쌓입니다. 2026년 흐름은 연합 + 파운데이션 모델로, LoRA 같은 작은 어댑터만 교환합니다.

## 이 모듈의 대표 논문

| 논문 | 핵심 |
|---|---|
| FedAvg (McMahan et al., 2017)[1] | 연합학습 창시, 가중평균 |
| FedProx (Li et al., 2020)[2] | proximal term으로 non-IID·straggler 대응 |
| Secure Aggregation (Bonawitz et al., 2017)[3] | 서버가 개별 업데이트를 못 보는 집계 |
| Ditto (Li et al., 2021)[4] | 정규화 개인화로 공정성·견고성 |
| FedBN (Li et al., 2021)[5] | BN을 로컬 유지해 feature non-IID 완화 |
| FederatedScope-LLM (Kuang et al., 2024)[6] | LLM 연합 파인튜닝 종합 패키지 |

## 마무리

이로써 심화편 태스크별 방법론 — 분류·회귀, 군집화, 차원축소·표현학습, 시계열, 그래프·GNN, 연합학습 — 이 모두 마무리됩니다. 각 트랙이 기초편의 한 주제를 받아 최신 방법론과 실무 트레이드오프까지 끌어올린 셈입니다.
