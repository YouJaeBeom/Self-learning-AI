# 태스크별 방법론 — 심화 트랙

목적(태스크)별 방법론을 방법론 갈래·변형·실무 트레이드오프로 깊이 보는 트랙입니다. 각 태스크는 대표 논문으로 다리를 놓고, 기초편의 한 주제를 받아 최신 방법론과 실무의 선택 기준까지 끌어올립니다.

## 여덟 태스크

| # | 태스크 | 심화에서 다루는 것 |
|:--:|---|---|
| 01 | [분류·회귀](./01_분류와_회귀/) | 부스팅 원리, 트리 앙상블 비교, 정형데이터 트리 대 딥러닝(TabPFN) |
| 02 | [군집화](./02_군집화/) | 밀도·스펙트럴·딥 군집화, 정답 없는 평가의 난점 |
| 03 | [차원축소·표현학습](./03_차원축소와_표현학습/) | t-SNE 대 UMAP, 자기지도 표현학습(대조·마스킹) |
| 04 | [시계열 분석](./04_시계열_분석/) | 딥러닝 예측(N-BEATS·PatchTST), 시계열 파운데이션 모델 |
| 05 | [이상치 탐지](./05_이상치_탐지/) | 밀도·고립·경계·딥·시계열 갈래 비교, 평가(PR-AUC) |
| 06 | [그래프·GNN](./06_그래프와_GNN/) | GCN·GraphSAGE·GAT, 귀납 대 변환, 과평활·그래프 트랜스포머 |
| 07 | [연합학습](./07_연합학습/) | non-IID·통신·개인화·보안 집계, 연합 + LLM |
| 08 | [컨티뉴얼 러닝](./08_컨티뉴얼_러닝/) | EWC·iCaRL·LwF 비교, 사전학습 모델 기반 CL |

각 트랙은 두세 편의 본문과 정리노트로 이뤄집니다. 공통 흐름은 기초편의 직관에서 출발해, 방법론이 어떤 가정 위에서 갈라지고 어디서 무너지는지를 비교하고, 최신 패러다임 전환(파운데이션 모델·in-context·제로샷)이 각 태스크를 어떻게 바꾸는지로 마무리하는 것입니다.
