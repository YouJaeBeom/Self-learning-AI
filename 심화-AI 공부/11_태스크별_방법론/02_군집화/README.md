# 02 · 군집화 (심화) — 둥근 무리를 넘어서

기초편의 k-means가 못 푸는 것에서 출발해, 군집화의 주요 변형들을 넓고 깊게 잡는 모듈입니다. 임의 모양·노이즈를 다루는 밀도 기반, 거리 너머를 보는 스펙트럴, 표현부터 배우는 딥 군집화, 그리고 정답 없는 학습의 고질적 난제인 평가까지 다룹니다. 전체를 관통하는 흐름은 k-means가 암묵적으로 깔고 있던 가정들을 하나씩 깨 나가는 것입니다.

기초편 02 군집화에서 k-means(중심 기반)·거리·실루엣을 잡고 오면 좋습니다. 이 모듈은 그 한계를 하나씩 공략합니다. 새 무거운 수학은 거의 없습니다. 핵심은 무엇을 닮음으로 볼 것인가의 발상 전환입니다.

## 이 모듈에서 답하는 질문

- k-means가 둥근 무리에만 통하는데, 초승달·반지 모양은 어떻게 묶나(밀도 기반)
- k를 모를 때, 노이즈(어디에도 안 속하는 점)는 어떻게 처리하나
- "거리"만으로 안 보이는 구조를 그래프로 바꿔 보면 무엇이 달라지나(스펙트럴)
- 픽셀·원문 같은 고차원 데이터는, 무엇을 재료(특징) 삼아 묶어야 하나(딥 군집화)
- 정답표가 없는데, "잘 묶었다"를 믿을 만하게 평가할 수 있나(평가의 난점)

한마디로 "k-means의 가정을 하나씩 깨면, 군집화는 어디까지 갈 수 있는가?"입니다.

## 이 모듈에 필요한 수학 (의미 수준)

밀도(이웃 반경 안 점의 개수)와 도달가능성의 직관(DBSCAN), 그래프 라플라시안과 고유벡터가 무엇을 드러내는지(스펙트럴, 계산이 아니라 의미), 대조학습 손실(같은 건 끌어당기고 다른 건 밀어내기)의 발상(딥 군집화)이면 충분합니다. 핵심은 무엇을 닮음으로 볼 것인가의 발상 전환입니다.
