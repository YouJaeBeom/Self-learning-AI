# 08 · 컨티뉴얼 러닝 (심화) — 세 갈래와 사전학습 전환

기초편의 세 갈래(정규화·재생·구조)를 대표 기법으로 정밀 비교하고(EWC·iCaRL·LwF), 2022년 이후의 큰 전환 — 사전학습 모델 기반 CL(prompt 기반, L2P) — 의 부상을 잡는 모듈입니다. 컨티뉴얼 러닝의 근본 문제는 치명적 망각이고, 세 갈래는 모두 무엇을 지켜서 그것을 막느냐의 다른 답입니다. 그리고 사전학습 백본의 등장이 이 문제의 풍경을 바꿉니다.

기초편 08 트랙에서 파국적 망각과 세 갈래의 직관을 잡고 오면 좋습니다. 새 모델 수학은 깊이 다루지 않습니다. 여기서는 세 갈래의 트레이드오프와 사전학습 전환이 주인공입니다.

## 이 모듈에서 답하는 질문

- EWC(정규화)·iCaRL(재생)·LwF(증류)는 무엇을 지키고 어디서 강하고 약한가
- "옛 데이터를 저장하느냐"가 왜 결정적 갈림길인가(rehearsal 대 rehearsal-free)
- 평가는 어떻게 하나, 평균 정확도·망각(forgetting)·시나리오는 무엇인가
- 거대 사전학습 모델이 있으면 CL이 어떻게 바뀌나, prompt 기반(L2P)은 왜 새로운가

## 이 모듈에 필요한 수학 (의미 수준)

피셔 정보로 가중치 중요도 추정(EWC), exemplar 선택과 NME(nearest-mean) 분류(iCaRL), 증류 손실(옛 모델 출력 따라 하기, LwF), prompt pool에서 키-쿼리로 prompt 선택(L2P), 평가 지표(평균 정확도, 망각/backward transfer)면 충분합니다. 여기서는 세 갈래의 트레이드오프와 사전학습 전환이 주인공입니다.
