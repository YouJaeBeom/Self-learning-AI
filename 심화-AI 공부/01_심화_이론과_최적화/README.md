# S1 · 심화 이론·최적화 — Adam과 일반화는 왜 작동하는가

기초편의 경사하강법 `θ ← θ − η·∇L`을 넘어, 현대 옵티마이저(Adam)와 정규화가 왜 작동하는지를 직관으로 푸는 모듈입니다. 더 나아가 "잘 외운 모델이 새 문제도 잘 푸는" 일반화의 비밀, 곧 평탄한 최소값까지 들여다봅니다.

이 모듈은 기초편 1단계(머신러닝 수학)의 최적화와 2단계의 일반화를 더 깊이 파고듭니다. 핵심 식 `θ ← θ − η·∇L`을 손에 쥐고 시작하면 됩니다. 새로 등장하는 수학은 개념 수준입니다. 헤시안·곡률 같은 용어가 나오지만 의미와 비유로만 다루고, 유도는 하지 않습니다.

## 이 모듈에서 답하는 질문

- 거의 모든 모델이 Adam으로 학습되는데, Adam은 경사하강법을 어떻게 개선한 것인가
- 1차 미분(기울기)만으로 충분한가, 2차 정보(곡률)는 무엇을 더 알려 주는가
- 학습률은 왜 스케줄·워밍업으로 다루는가, L2·드롭아웃은 왜 과적합을 줄이는가
- 같은 손실이라도 '평탄한' 최소값이 왜 더 잘 일반화하는가

한마디로 이 모듈의 핵심 질문은 "학습을 안정시키고 일반화시키는 보이지 않는 손은 무엇인가?"입니다.

## 이 모듈에 필요한 배경

기초편 1단계의 경사하강법·연쇄법칙·기울기 ∇L, 2단계의 과적합·편향–분산·정규화를 전제합니다. 새 수학은 개념 수준이며, 헤시안·곡률 같은 용어가 나오더라도 의미와 비유로만 다루므로 유도를 따라갈 필요는 없습니다.
