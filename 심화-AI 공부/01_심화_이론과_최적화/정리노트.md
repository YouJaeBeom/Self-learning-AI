# S1 정리노트 — 한눈에 보는 심화 이론·최적화

이 모듈의 본문을 한 장으로 압축했습니다. 네 글의 핵심을 다시 짚고, 큰 연결 고리와 대표 논문, 그리고 다음 모듈로 가는 다리를 정리합니다.

## 네 글의 요지

| 주제 | 왜 필요한가 | 핵심 한 문장 |
|---|---|---|
| 옵티마이저 심화 | 경사하강의 두 약점을 고치려고 | Adam = 모멘텀(방향 관성, 1차) + 적응적 보폭(손잡이별, 2차). "방향을 험함으로 나눈다" |
| 2차 정보·자연경사 | 기울기가 못 보는 걸 보려고 | 곡률(헤시안) = 기울기의 변화율. LLM엔 너무 커서 Adam이 값싼 근사로 정신만 빌림 |
| 학습률·정규화 | 보폭과 겸손함을 조율하려고 | 워밍업 → 감쇠로 보폭 조절. L2 정규화 = 가중치에 정규분포 사전분포(MAP) |
| 일반화 이론 | "왜 새 문제도 푸나"에 답하려고 | 평탄한 최소값 = 흔들림에 강한 해 → 새 데이터에 버팀. 정규화·BN이 풍경을 평탄하게 |

## 이 모듈의 핵심 수식

| 수식 | 한 줄 의미 |
|---|---|
| `m←β₁m+(1−β₁)∇L`, `v←β₂v+(1−β₂)(∇L)²`, `θ←θ−η·m/(√v+ε)` | Adam = 방향(m)을 험함(√v)으로 나눈 갱신 |
| `H = ∇²L` | 헤시안 = 곡률 = 기울기의 변화율(방향별 굽음) |
| `L_전체 = L_데이터 + λ‖θ‖²` | L2 정규화: 가중치 크기를 벌점으로 |
| `θ* = argmax[ log P(D\|θ) + log P(θ) ]` | MAP: MLE + 사전분포 → L2와 같음 |
| `평탄함 ↔ 헤시안 고유값 작음` | 평탄한 최소값 = 곡률이 작은 넓은 분지 |

이 모듈을 관통하는 큰 연결은 세 가지입니다. 정규화는 곧 사전분포(확률론)이고, Adam은 2차 정보의 값싼 근사이며, 평탄한 최소값은 일반화의 단서입니다. 셋 다 기초편 수학(베이즈·곡률·비볼록)을 한 층 더 깊이 끌어올린 것입니다.

## 이 장의 대표 논문

| 주제 | 논문 | 핵심 한 줄 |
|---|---|---|
| 옵티마이저 | Adam (Kingma & Ba, 2014)[1] | 모멘텀+적응적 보폭, 딥러닝 기본 옵티마이저 |
| 옵티마이저 | AdamW (Loshchilov & Hutter, 2017)[2] | 가중치 감쇠 분리, LLM 표준 (2026-06 기준) |
| 워밍업 | RAdam (Liu et al., 2019)[3] | 초기 적응 추정의 분산 문제 → 워밍업 근거 |
| 정규화 | Batch Normalization (Ioffe & Szegedy, 2015)[4] | 층 분포를 다듬어 학습 안정·풍경 평탄화 |
| 일반화 | Layer Normalization (Ba et al., 2016)[5] | 샘플 내부 정규화 → 트랜스포머 표준 |
| 일반화 | Large-Batch Training / sharp minima (Keskar et al., 2017)[6] | 큰 배치가 날카로운 최소값에 빠지는 경향 |

## 다음 모듈 예고 — S2 · 생성모델 계보

지금까지(기초편 포함) 다룬 모델은 대부분 분류·예측이었습니다. 입력을 받아 라벨이나 값을 맞히는 일이었죠. S2는 방향을 틉니다. 질문이 "이 입력의 정답은?"에서 "이런 데이터를 새로 만들어 낼 수 있나?"로 바뀝니다. 오토인코더 → VAE → GAN → Diffusion → Flow Matching으로 이어지는 생성모델의 계보를 따라가며, 이미지·영상 생성의 원리를 봅니다. 첫 대표 논문은 VAE(Kingma & Welling, 2013), GAN(Goodfellow et al., 2014), DDPM(Ho et al., 2020)입니다. S1에서 다진 확률·정규화 감각이 그대로 쓰입니다.
