# S4 · LLM 사전학습 심화 — 수천 GPU로 모델 하나 키우기

기초편 5단계의 "다음 토큰 예측"을, 수천 GPU와 수조 토큰 규모로 실제 실현하는 공학으로 구체화하는 모듈입니다. 데이터 큐레이션, 분산학습, Chinchilla 최적(데이터 대 파라미터 균형), 실전 레시피를 한눈에 잡습니다. 모델의 학습 목표 자체는 기초편과 똑같습니다. 달라지는 것은 그 단순한 목표를 거대한 규모에서 돌리기 위한 받침대들입니다.

기초편 5단계의 '사전학습 규모'를 실제 엔지니어링으로 내려 보므로, 다음 토큰 예측과 스케일링 감각을 쥐고 시작하면 좋습니다. 새 수학은 없습니다. 공학(병렬화·메모리·비용) 개념이 새로 등장하지만 비유로만 다룹니다.

## 이 모듈에서 답하는 질문

- 사전학습에 무엇을, 어떻게 먹이나(데이터 수집·필터링·중복 제거·토크나이저)
- 모델 하나를 왜 한 GPU로 못 학습시키나, 분산학습(데이터·텐서·파이프라인 병렬, ZeRO)은 무엇인가
- 주어진 연산 예산에서 모델 크기와 데이터 양을 어떻게 나눠야 최선인가(Chinchilla 최적)
- 거대 학습을 가능케 하는 실전 레시피(혼합정밀·체크포인팅)는 무엇인가

한마디로 이 모듈의 핵심 질문은 "다음 토큰 예측을, 수천 GPU 규모로 어떻게 실제로 돌리는가?"입니다.
