# S3 · 트랜스포머 심화 — 어텐션을 더 빠르고 길게

기초편 어텐션 `softmax(QKᵀ/√dₖ)·V`의 실전 한계(O(n²) 비용과 문맥 길이)를 정면으로 뚫는 모듈입니다. FlashAttention(메모리 병목), RoPE·ALiBi(위치), KV 캐시(긴 문맥)의 핵심 아이디어를 잡습니다. 이 셋은 따로 노는 기법이 아니라, 하나의 적인 O(n²)을 서로 다른 각도에서 공격하는 한 팀입니다.

기초편 4단계의 어텐션을 공학적 한계까지 밀어붙이므로, `softmax(QKᵀ/√dₖ)·V`를 손에 쥐고 시작하면 좋습니다. 새 수학은 없습니다. 비용(복잡도)과 하드웨어 감각이 새로 등장하지만 비유로만 다룹니다.

## 이 모듈에서 답하는 질문

- 어텐션은 왜 문맥이 길어지면 비용이 제곱(O(n²))으로 폭증하는가
- FlashAttention은 어떻게 결과를 바꾸지 않고 메모리 병목만 풀었나
- 위치를 절대 좌표 대신 상대 관계로 주면 무엇이 좋아지나(RoPE·ALiBi)
- KV 캐시는 무엇이고, 긴 문맥을 다루는 아이디어는 무엇인가

한마디로 이 모듈의 핵심 질문은 "같은 어텐션을, 어떻게 더 빠르고 더 길게 돌리는가?"입니다.
