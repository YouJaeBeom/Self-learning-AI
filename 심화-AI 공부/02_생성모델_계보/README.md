# S2 · 생성모델 계보 — 분류를 넘어 생성으로

"입력의 정답 맞히기"(분류)를 넘어, "이런 데이터를 새로 만들어 내기"(생성)의 세계로 들어가는 모듈입니다. 오토인코더 → VAE → GAN → Diffusion → Flow Matching으로 이어지는 계보를, 이미지 생성을 실마리로 따라갑니다. 이 다섯 갈래는 서로 경쟁하면서도 아이디어를 주고받으며 발전했고, 그 흐름을 따라가다 보면 오늘날 텍스트→이미지 시스템의 내부가 자연스럽게 보입니다.

기초편 3단계의 신경망 위에서 데이터 분포 자체를 배우는 모델을 다룹니다. 특히 S1에서 다진 확률·정규화 감각, 그중에서도 정규분포 사전분포(MAP) 직관이 VAE에서 곧장 다시 쓰입니다. 새 수학은 개념 수준입니다. ELBO·점수(score) 같은 용어가 나오지만 의미와 비유로만 다루고 유도는 하지 않습니다.

## 이 모듈에서 답하는 질문

- '분류'와 '생성'은 무엇이 근본적으로 다른가, '데이터 분포를 배운다'는 게 무슨 뜻인가
- VAE의 잠재변수와 ELBO는 무엇을 하려는 것인가
- GAN의 '생성자 대 판별자' 경쟁은 어떻게 진짜 같은 이미지를 만드나
- 요즘 이미지·영상 생성의 주역 Diffusion은 노이즈를 제거해 어떻게 그림을 그리나
- Flow Matching은 Diffusion을 어떻게 더 단순하게 만들었나

한마디로 이 모듈의 핵심 질문은 "기계는 어떻게 '없던 데이터'를 그럴듯하게 만들어 내는가?"입니다.

기초편 1단계의 확률분포(정규분포)·교차엔트로피, 3단계의 신경망·인코더/디코더 감각을 전제합니다. S1의 정규분포 사전분포(MAP) 직관이 VAE에서 곧장 쓰이므로, 그 감각을 챙겨 오면 좋습니다. 앞서 말했듯 ELBO·점수 같은 용어는 의미와 비유로만 다루므로 유도를 따라갈 필요는 없습니다.
