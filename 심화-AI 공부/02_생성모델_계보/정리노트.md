# S2 정리노트 — 한눈에 보는 생성모델 계보

이 모듈의 본문을 한 장으로 압축했습니다. 네 갈래의 핵심을 다시 짚고, 계보의 공통 골격과 대표 논문, 다음 모듈로 가는 다리를 정리합니다.

## 네 갈래의 요지

| 주제 | 왜 필요한가 | 핵심 한 문장 |
|---|---|---|
| 오토인코더·VAE | 분류를 넘어 분포를 배우려고 | VAE = 정규분포 잠재공간으로 압축·복원 → 그 공간에서 뽑아 생성. ELBO = 복원 + KL(사전분포) |
| GAN | '진짜 같은가'를 직접 겨루려고 | 생성자 대 판별자 경쟁(위조범 대 경찰). 날카롭지만 불안정·모드 붕괴 |
| Diffusion | 어려운 생성을 쉬운 단계로 쪼개려고 | 노이즈를 더했다가 '조금씩 걷어내기'를 배워 노이즈에서 그림을 피움. 느린 게 약점 |
| Flow Matching | 느린 경로를 곧게 펴려고 | 노이즈→데이터를 직선 경로로, '가야 할 속도'를 맞힘 → 적은 단계로 빠르게 |

## 이 모듈의 핵심 수식

| 수식 | 한 줄 의미 |
|---|---|
| `ELBO = 복원 항 − KL 항` | VAE: 잘 복원 + 잠재공간을 표준정규로(사전분포) |
| `min_G max_D [진짜판정+가짜판정]` | GAN: 한 점수를 두고 생성자·판별자가 반대로 줄다리기 |
| `‖넣은 노이즈 − 예측 노이즈‖²` | Diffusion: '낀 노이즈 맞히기'(MSE)가 학습 목표 |
| `∇ log p(x)` (점수) | 데이터가 더 그럴듯해지는 방향 = 노이즈 제거 방향 |
| `‖목표 속도 − 예측 속도‖²` | Flow Matching: '가야 할 속도 맞히기'(직선 경로) |

이 계보의 공통 골격은 하나입니다. 간단한 분포(노이즈)에서 복잡한 데이터로 변환한다는 것. VAE가 그 원형을 만들었고, Diffusion이 변환을 점진화했으며, Flow Matching이 경로를 직선화했습니다. 모두 S1의 확률·정규분포 사전분포 감각 위에 서 있습니다.

## 이 장의 대표 논문

| 주제 | 논문 | 핵심 한 줄 |
|---|---|---|
| VAE | VAE (Kingma & Welling, 2013)[1] | 잠재변수를 분포로, ELBO로 학습 |
| GAN | GAN (Goodfellow et al., 2014)[2] | 생성자–판별자 적대 학습 |
| GAN | StyleGAN (Karras et al., 2019)[3] | 고품질 얼굴 생성, GAN의 정점 |
| Diffusion | DDPM (Ho et al., 2020)[4] | 노이즈 예측이라는 단순 목표로 고품질 생성 |
| Diffusion | Score-based SDE (Song et al., 2021)[5] | 노이즈 제거 = 점수 추정으로 통합 |
| Diffusion | Latent Diffusion (Rombach et al., 2022)[6] | VAE 잠재공간 확산, 텍스트→이미지 대중화 |
| Flow | Flow Matching (Lipman et al., 2022)[7] | 속도장 회귀로 생성, Diffusion 일반화 |
| Flow | Rectified Flow (Liu et al., 2022)[8] | 경로 직선화로 빠른 생성 |

2026-06 기준 이미지·영상 생성의 주류는 Diffusion·Flow 계열입니다. 특정 승자보다 계보를 기억하는 것이 오래간다는 S0의 교훈을 다시 새겨 둘 만합니다.

## 다음 모듈 예고 — S3 · 트랜스포머 심화

S2에서 생성모델 계보를 봤습니다. 이제 LLM 라인으로 돌아가, 기초편 4단계의 트랜스포머를 실전 한계까지 파고듭니다. 기초편에서 어텐션 `softmax(QKᵀ/√dₖ)·V`를 배웠는데, 이 어텐션은 문맥이 길어지면 비용이 제곱으로 폭증합니다(O(n²)). 왜 그런지, 그리고 어떻게 뚫는지를 다룹니다. FlashAttention(메모리 병목 풀기), RoPE·ALiBi(위치를 더 똑똑하게), KV 캐시(긴 문맥)가 주제입니다. 첫 대표 논문은 FlashAttention(Dao et al., 2022), RoPE(Su et al., 2021), ALiBi(Press et al., 2021)입니다.
