# AI 자습 노트 — 원리부터 최신 논문까지

> 혼자 AI를 공부하며 **개념·이론·수학·논문**을 스스로 정리한 노트 모음입니다.
> 코드 구현이 아니라 **"왜 그렇게 되는가"의 원리**를 말로 설명할 수 있게 되는 것이 목표입니다.

이 저장소는 한 권이 아니라 **3부작(三部作)** 으로 이뤄집니다. 원리를 잡는 **기초편**, 응용과 최신 흐름을 읽는 **심화편**, 그리고 그 원리를 만든 논문 원전으로 들어가는 **핵심논문편(부록)** 입니다. 세 책은 같은 직관·같은 톤으로 이어지도록 썼습니다.

*모든 내용은 자습용 정리 노트이며, 최신 기법 판단은 작성 시점(2026-06) 기준입니다.*

## 🧭 어떻게 이어지나

세 책은 **난이도의 3단계**이자, **같은 주제를 세 깊이로** 읽는 구조입니다.

```
원리를 이해한다      →   응용·최신을 읽는다      →   원전으로 들어간다
   ① 기초편                  ② 심화편                  ③ 핵심논문편
 (개념·직관·수학)      (방법론 심화 + 논문 다리)      (바이블 → 최신 논문)
```

특히 **태스크별 방법론**(분류·군집화·차원축소·시계열·이상치·그래프·연합학습·컨티뉴얼)은
세 책에 각각 한 폴더씩 있어, 한 태스크를 **기초 → 심화 → 논문** 세 단계로 따라갈 수 있습니다.

## 🚶 추천 학습 경로

1. **처음이라면** — [기초편](기초-AI%20공부/README.md)을 0단계부터 순서대로. 각 장의 `README → 본문(번호순) → 정리노트` 흐름을 지킵니다.
2. **원리가 잡혔다면** — [심화편](심화-AI%20공부/README.md)에서 관심 모듈(예: 생성모델·RAG·에이전트)로. 유도·증명은 원문에 맡기고 "왜 중요한가"를 잡습니다.
3. **더 깊이 파려면** — [핵심논문편](부록-AI%20핵심논문/README.md)에서 해당 테마의 바이블과 최신작을 나란히. 이 노트는 원문으로 가는 **다리**입니다.
4. **목적이 분명하다면** — 태스크(예: 시계열)를 골라 세 책의 같은 트랙을 세로로 훑습니다.

## ✍️ 이 노트를 쓴 원칙

- **설명할 수 있어야 안다** — 비전공자에게 말로 풀 수 있을 때 통과.
- **수식은 의미부터** — 기호를 읽기 전에 "이 식이 무엇을 최소/최대화하려는가"를 먼저 묻는다.
- **직관 → 원전** — 개념의 직관과 연결을 먼저 잡고, 깊은 증명·예제는 외부 자료와 원문으로 채운다.
- **정직한 인용** — 논문은 제목·저자·연도·venue·arXiv ID를 웹으로 교차검증한다 (부록편).

> 목표는 '진도 완주'가 아니라 **'원리를 말로 설명할 수 있게 되는 것'** 입니다.
> 수학 앞에서 멈추지 말고, 의미부터 천천히 — 그러나 꾸준히.
